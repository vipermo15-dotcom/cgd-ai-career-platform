# Week 06 예제 - 포트폴리오 기획 실전 예시

## 예제 1: 브랜드 디자이너 포트폴리오 목차

```
김민준 Brand Design Portfolio 2026
총 24페이지

01. Cover               (1p) - 이름, 직군, 한 줄 슬로건
02. About Me            (2p) - 자기소개 + 스킬 인포그래픽
03. Project 01          (5p) - 한국茶 브랜드 아이덴티티
    - 브리프 + 리서치
    - 무드보드 + 컨셉
    - 로고 개발 과정
    - 최종 어플리케이션
04. Project 02          (5p) - 친환경 스타트업 CI
05. Project 03          (4p) - 로컬 카페 패키지
06. Project 04          (4p) - 학교 홍보물 시리즈
07. Skills              (1p) - Adobe CC, AI 도구
08. Contact             (1p) - 연락처 + QR코드
09. Thank you           (1p) - 마무리 인사
```

## 예제 2: UI/UX 디자이너 포트폴리오 목차

```
이지현 UX Design Portfolio 2026
총 32페이지

01. Cover               (1p)
02. Introduction        (2p) - UX 철학 + 프로세스 소개
03. Project 01          (8p) - 배달앱 UX 개선
    - 문제 정의
    - 사용자 리서치
    - 페르소나
    - 와이어프레임
    - 프로토타입
    - 결과 측정
04. Project 02          (7p) - 도서관 앱 신규 기획
05. Project 03          (6p) - 학교 홈페이지 리디자인
06. Design System       (3p) - 컴포넌트 & 스타일 가이드
07. Skills              (2p) - Figma, Protopie, 리서치 방법론
08. Contact             (1p)
09. References          (2p) - 참고 출처
```
