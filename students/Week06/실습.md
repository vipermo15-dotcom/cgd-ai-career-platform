## 이번 주 실습 목표
취업에 특화된 포트폴리오 전략을 수립하고 목차와 작업물 구성을 완성합니다.

## 실습 1: 나의 직군 분석

### 직군별 포트폴리오 특성

| 직군 | 중점 작업물 | 분량 | 핵심 어필 |
|------|------------|------|----------|
| 브랜드 디자이너 | 로고, CI/BI, 편집 | 20~30p | 스타일 일관성 |
| UI/UX 디자이너 | 앱, 웹, 프로토타입 | 30~50p | 사용자 경험 |
| 콘텐츠 디자이너 | SNS, 배너, 영상 | 20~30p | 다양성 + 속도 |
| 영상 편집자 | 영상 릴, 유튜브 | 링크 위주 | 크리에이티브 |
| 패키지 디자이너 | 패키지, 3D | 20~25p | 입체감, 인쇄 |

## 실습 2: 작업물 인벤토리 작성

보유한 모든 작업물 목록을 만들어보세요:

```markdown
## 내 작업물 인벤토리

### 학교 과제
1. [작업명] - [과목] - [연도] - 파일 위치: [경로]
2. [작업명] - [과목] - [연도] - 파일 위치: [경로]
...

### 개인 프로젝트
1. [작업명] - [분야] - [연도] - 파일 위치: [경로]
...

### 외부 활동 (공모전, 봉사 등)
1. [작업명] - [기관] - [연도] - 파일 위치: [경로]
...
```

## 실습 3: 포트폴리오 선별 기준

### BEST 3 선별 기준
```
[ ] 완성도가 높은가? (미완성 작업 제외)
[ ] 내 직군과 관련 있는가?
[ ] 제작 과정을 설명할 수 있는가?
[ ] 차별화 요소가 있는가?
[ ] 최근 1~2년 작업인가? (구형 스타일 주의)
```

## 실습 4: 포트폴리오 목차 설계

```markdown
## [이름] 포트폴리오 목차 초안

### Cover (표지) - 1p
- 이름, 직군, 연락처, 한 줄 소개

### Introduction (자기소개) - 1~2p
- 사진 또는 일러스트
- 한 줄 철학
- 보유 스킬 인포그래픽

### Project 01: [제목] - 4~6p
- 프로젝트 개요 (1p)
- 리서치 과정 (1p)
- 디자인 과정 (1~2p)
- 최종 결과물 (1~2p)

### Project 02: [제목] - 4~6p
[위와 동일 구조]

### Project 03: [제목] - 4~6p
[위와 동일 구조]

### Skills & Tools - 1p
- Adobe 제품군
- AI 도구
- 기타 툴

### Contact - 1p
- 이메일, 전화, SNS, Behance

총 예상 페이지: 18~25p
```
