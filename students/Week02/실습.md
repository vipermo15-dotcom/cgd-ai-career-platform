# Week 02 실습 - VSCode + Markdown 기초

## 이번 주 실습 목표
VS Code의 핵심 기능을 익히고, Markdown으로 전문적인 문서를 작성합니다.

---

## 실습 1: VS Code 필수 설정

### 추천 확장 프로그램 설치

```
Extensions 탭 (Ctrl+Shift+X) 에서 검색하여 설치:

1. Markdown Preview Enhanced - Markdown 실시간 미리보기
2. Korean Language Pack - 한국어 인터페이스
3. GitLens - Git 기능 강화
4. Prettier - 코드/문서 자동 정렬
5. Material Icon Theme - 파일 아이콘 개선
```

### 유용한 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+Shift+P` | 명령 팔레트 열기 |
| `Ctrl+\`` | 터미널 열기 |
| `Ctrl+Shift+V` | Markdown 미리보기 |
| `Ctrl+K V` | 사이드에서 미리보기 |
| `Alt+Z` | 줄 바꿈 토글 |
| `Ctrl+/` | 줄 주석 처리 |

## 실습 2: Markdown 핵심 문법

### 제목 (Heading)

```markdown
# H1 - 가장 큰 제목 (문서 제목)
## H2 - 두 번째 제목 (챕터)
### H3 - 세 번째 제목 (섹션)
#### H4 - 네 번째 제목 (소섹션)
```

### 텍스트 서식

```markdown
**굵게** (Bold)
*기울기* (Italic)
~~취소선~~
`코드` (인라인 코드)
> 인용문 블록

일반 텍스트
줄바꿈은 빈 줄 하나를 추가합니다

---  ← 수평선
```

### 목록

```markdown
## 순서 없는 목록
- 항목 1
- 항목 2
  - 하위 항목 (스페이스 2칸)
  - 하위 항목
- 항목 3

## 순서 있는 목록
1. 첫 번째
2. 두 번째
3. 세 번째

## 체크박스 목록
- [x] 완료된 항목
- [ ] 미완료 항목
- [ ] 미완료 항목
```

### 링크와 이미지

```markdown
[링크 텍스트](https://url.com)
[GitHub 바로가기](https://github.com)

![이미지 설명](이미지URL 또는 경로)
![포트폴리오 표지](images/cover.jpg)
```

### 표 (Table)

```markdown
| 열1 | 열2 | 열3 |
|-----|-----|-----|
| 셀1 | 셀2 | 셀3 |
| 셀4 | 셀5 | 셀6 |

정렬:
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
|:----------|:-----------:|------------:|
| 내용      |    내용     |          내용 |
```

### 코드 블록

````markdown
```언어명
코드 내용
```

예시:
```python
print("Hello, CGD!")
```

```bash
git commit -m "커밋 메시지"
```
````

## 실습 3: 자기소개 문서 작성

다음 템플릿을 활용하여 자신의 자기소개 Markdown 문서를 작성하세요:

```markdown
# [이름]의 디자이너 이야기

## 자기소개

> "한 줄로 나를 표현한다면..."

안녕하세요! 저는 [지역]에서 [전공]을 공부하고 있는 [이름]입니다.

## 나의 강점

- 🎨 **창의력**: [구체적인 예시]
- 💪 **성실함**: [구체적인 예시]  
- 🤝 **협업 능력**: [구체적인 예시]

## 보유 스킬

| 분야 | 도구 | 숙련도 |
|------|------|--------|
| 그래픽 | Photoshop | ★★★★☆ |
| 벡터 | Illustrator | ★★★☆☆ |
| 영상 | Premiere Pro | ★★★☆☆ |

## 대표 작업

### 1. [프로젝트명]
- **목적**: [제작 목적]
- **기간**: [제작 기간]
- **역할**: [내가 한 일]
- **결과**: [성과/배운 점]

## 앞으로의 목표

1. [단기 목표 - 3개월]
2. [중기 목표 - 6개월]
3. [장기 목표 - 1년]

## 연락처
- 이메일: your@email.com
- Behance: behance.net/yourprofile
```

## 실습 4: Markdown을 GitHub에 올리기

```bash
# 자기소개 파일 만들기
touch about-me.md

# VS Code로 열어서 작성
code about-me.md

# 작성 후 GitHub에 업로드
git add about-me.md
git commit -m "docs: 자기소개 Markdown 문서 추가"
git push
```
