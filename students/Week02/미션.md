# Week 02 미션 - VSCode + Markdown

## 미션 개요

VS Code에서 Markdown을 작성하고, 완성된 문서를 GitHub에 업로드합니다.

---

## 필수 미션

### 미션 1: VS Code 확장 프로그램 5개 설치
```
[ ] Markdown Preview Enhanced
[ ] Korean Language Pack
[ ] GitLens
[ ] Prettier
[ ] 본인이 선택한 1개 추가
[ ] 캡처 제출 (Extensions 설치된 화면)
```

### 미션 2: Markdown 문법 연습 파일 작성
`markdown-practice.md` 파일을 만들어 다음 모든 문법을 사용하세요:
```
[ ] 제목 (H1~H4) 사용
[ ] 굵게, 기울기, 취소선 사용
[ ] 순서없는 목록 + 체크박스
[ ] 순서있는 목록
[ ] 표(Table) - 최소 3열 3행
[ ] 코드 블록 (언어 지정)
[ ] 링크와 이미지
[ ] 인용문 블록
[ ] 수평선
```

### 미션 3: 자기소개 전문 문서 작성
`about-me.md` 파일을 작성하고 GitHub에 업로드:
```
[ ] 제목: "나는 어떤 디자이너인가"
[ ] 자기소개 단락 (3문단 이상)
[ ] 보유 스킬 표
[ ] 대표 작업 섹션 (1개 이상)
[ ] 향후 목표 (단기/중기/장기)
[ ] 연락처 섹션
```
