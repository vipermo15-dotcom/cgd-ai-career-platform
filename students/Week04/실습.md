# Week 04 실습 - MCP 설치 + GitHub 연동

## 이번 주 실습 목표
Claude Desktop에 MCP를 설치하고, GitHub와 연동하여 AI가 내 저장소를 직접 읽고 수정할 수 있게 합니다.

---

## 실습 1: Claude Desktop 설치

1. https://claude.ai/download 접속
2. macOS 또는 Windows 버전 다운로드
3. 설치 후 로그인 (claude.ai 계정 사용)

---

## 실습 2: MCP 서버란?

MCP(Model Context Protocol)는 Claude가 외부 도구와 연결할 수 있게 해주는 프로토콜입니다.

```
Claude Desktop
    │
    ├── GitHub MCP → 저장소 읽기/쓰기
    ├── Firecrawl MCP → 웹 스크래핑
    ├── Filesystem MCP → 로컬 파일 접근
    └── 기타 MCP 서버들...
```

---

## 실습 3: GitHub MCP 설치

### Node.js 설치 확인
```bash
node --version  # v18 이상 필요
npm --version
```

### MCP 설정 파일 위치
- Mac: `~/Library/Application Support/Claude/claude_desktop_config.json`
- Windows: `%APPDATA%\Claude\claude_desktop_config.json`

### 설정 파일 작성
```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-github"
      ],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_your_token_here"
      }
    }
  }
}
```

---

## 실습 4: GitHub Personal Access Token 발급

1. GitHub.com 로그인
2. Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
3. "Generate new token (classic)" 클릭
4. 권한 선택:
   - [x] repo (전체 저장소 접근)
   - [x] read:org (조직 읽기)
5. 생성된 토큰 복사 (한 번만 보임!)
6. 설정 파일의 `GITHUB_PERSONAL_ACCESS_TOKEN` 에 붙여넣기

---

## 실습 5: MCP 연동 테스트

Claude Desktop 재시작 후:

```
Claude에게 질문:
"내 GitHub 저장소 목록을 보여줘"

"portfolio-2026 저장소의 README.md 내용을 읽어줘"

"portfolio-2026 저장소에 새 파일 test.md를 만들어줘"
```

---

## 실습 6: Filesystem MCP 추가 (로컬 파일 접근)

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_your_token"
      }
    },
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/Users/내이름/Desktop/portfolio"
      ]
    }
  }
}
```
