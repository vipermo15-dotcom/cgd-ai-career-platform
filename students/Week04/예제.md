# Week 04 예제 - MCP 활용 실전 예시

## 예제 1: Claude + MCP로 포트폴리오 저장소 관리

### 대화 예시

**사용자**: "내 portfolio-2026 저장소를 분석하고 개선 방향을 알려줘"

**Claude (MCP 활용)**:
```
GitHub MCP로 저장소를 확인했습니다.

현재 파일 구조:
- README.md (기본 소개만 있음)
- 커밋: 3개 (너무 적음)

개선 방향:
1. projects/ 폴더 생성하여 작업물 정리
2. 각 프로젝트별 README.md 추가
3. GitHub Pages 설정으로 웹 포트폴리오 구현
4. 커밋을 주 3회 이상 유지

바로 projects/ 폴더 구조를 만들어드릴까요?
```

## 예제 3: MCP 오류 해결 모음

| 오류 | 원인 | 해결 |
|------|------|------|
| "MCP disconnected" | 설정 오류 | JSON 문법 확인 |
| "401 Unauthorized" | PAT 권한 부족 | repo 권한 추가 |
| "npx not found" | Node.js 미설치 | nodejs.org에서 설치 |
| 서버가 안 보임 | 재시작 안 함 | Claude Desktop 완전 재시작 |
