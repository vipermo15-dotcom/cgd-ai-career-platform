# Week 03 미션 - Claude 무료 계정 + 기본 프롬프트

## 필수 미션

### 미션 1: Claude 계정 가입 + Project 설정
```
[ ] claude.ai 가입 완료
[ ] "취업 준비 2026" Project 생성
[ ] Instructions에 나의 정보 입력
[ ] 캡처 제출
```

### 미션 2: 프롬프트 5개 작성 및 결과 저장
다음 5가지 주제로 Claude에게 질문하고, 결과를 `prompts-result.md`에 저장:

```
[ ] 1. 내 포트폴리오 목차 피드백 받기
[ ] 2. 목표 기업 1곳 분석 요청
[ ] 3. 자기소개서 도입부 초안 받기
[ ] 4. 내 직군의 채용 트렌드 분석
[ ] 5. 면접 예상 질문 10개 요청
```

저장 형식:
```markdown
## 프롬프트 1: 포트폴리오 목차 피드백

**내가 입력한 프롬프트:**
[프롬프트 전문]

**Claude 답변:**
[답변 전문]

**느낀 점 / 활용 계획:**
[본인 의견]
```

### 미션 3: 프롬프트 개선하기
미션 2의 프롬프트 중 1개를 선택하여:
```
[ ] 처음 프롬프트 작성
[ ] Claude 답변 확인
[ ] 프롬프트 개선 (더 구체적으로)
[ ] 개선 후 답변 비교
[ ] 두 답변의 차이 분석 작성
```
