# Week 03 실습 - Claude 무료 계정 + 기본 프롬프트

## 이번 주 실습 목표
Claude AI에 가입하고, 효과적인 프롬프트 작성법을 익힙니다.

---

## 실습 1: Claude 계정 가입

### 가입 절차
1. claude.ai 접속
2. "Sign up" 클릭
3. 이메일 또는 Google 계정으로 가입
4. 이메일 인증 완료
5. 기본 정보 입력

### 무료 플랜 한도
- 하루 약 10~20회 대화 가능
- Claude 3 Haiku 모델 사용
- Projects 기능 제한적 사용

---

## 실습 2: Claude Projects 설정

Projects는 Claude에게 나만의 맥락을 저장하는 공간입니다.

### 취업 준비용 Project 만들기
1. claude.ai 접속 후 좌측 "Projects" 클릭
2. "+ New project" 클릭
3. 이름: "취업 준비 2026"
4. Instructions (지시사항) 입력:

```
나는 컴퓨터그래픽디자인 전공 졸업 예정자입니다.
목표 직군: 그래픽 디자이너, UI/UX 디자이너
보유 스킬: Photoshop(중급), Illustrator(초급), Premiere Pro(초급)
학교: [학교명]
졸업 예정: 2026년 2월

이 프로젝트에서는 취업 준비와 포트폴리오 제작을 도와주세요.
항상 한국어로 답변해주세요.
구체적이고 실용적인 조언을 주세요.
```

## 실습 3: 좋은 프롬프트 작성법

### CLEAR 프레임워크

```
C - Context (맥락): 내가 누구인지, 상황이 무엇인지
L - Length (길이): 원하는 답변 분량
E - Examples (예시): 원하는 형식의 예시
A - Action (행동): 구체적으로 무엇을 해달라고 하는지
R - Role (역할): Claude에게 어떤 역할을 부여하는지
```

### 나쁜 프롬프트 vs 좋은 프롬프트

**나쁜 예시:**
```
포트폴리오 좀 봐줘
```

**좋은 예시:**
```
당신은 10년 경력의 시니어 그래픽 디자이너입니다.
저는 컴퓨터그래픽디자인 전공 신입생으로 첫 포트폴리오를 준비 중입니다.
목표 직군은 브랜드 디자이너입니다.

다음 포트폴리오 목차를 검토하고 피드백해주세요:
1. 표지
2. 자기소개
3. 브랜드 아이덴티티 프로젝트 (3개)
4. 패키지 디자인 (2개)
5. 연락처

피드백 항목:
- 목차 구성의 적절성
- 추가하면 좋을 섹션
- 각 섹션의 적정 페이지 수 추천
- 브랜드 디자이너 취업에 특화된 조언

500자 이내로 핵심만 답변해주세요.
```

## 실습 4: 다양한 프롬프트 직접 써보기

### 프롬프트 1: 포트폴리오 피드백
```
나는 그래픽 디자이너 지망생입니다.
[작업물 설명 또는 링크 붙여넣기]

이 작업물을 취업 면접관 관점에서 평가해주세요:
1. 강점 3가지
2. 개선점 3가지
3. 추가하면 좋은 요소
4. 이 작업물이 어울리는 직군/기업 유형
```

### 프롬프트 2: 자기소개서 초안
```
컴퓨터그래픽디자인 전공자의 자기소개서를 작성해주세요.

지원 기업: 카카오
지원 직군: UX 디자이너
나의 경험: [경험 적기]
나의 강점: 사용자 중심 사고, 팀워크
분량: 800자 내외
```

### 프롬프트 3: 기업 분석
```
네이버에 UX 디자이너로 지원하려 합니다.
신입 지원자 관점에서 다음을 분석해주세요:

1. 네이버 UX팀이 추구하는 디자인 방향
2. 신입에게 기대하는 역량 TOP 5
3. 경쟁력 있는 포트폴리오 구성 방법
4. 자기소개서에 반드시 넣어야 할 키워드
5. 예상 면접 질문 5개
```

## 실습 5: Claude와의 대화 팁

```
1. 역할 부여하기: "당신은 ~~입니다"
2. 구체적인 수치 제시: "3가지", "500자", "1페이지"
3. 형식 지정하기: "표로", "번호 목록으로", "JSON으로"
4. 예시 제공하기: "다음 예시처럼..."
5. 피드백 반복하기: "좀 더 구체적으로", "더 캐주얼하게"
```
