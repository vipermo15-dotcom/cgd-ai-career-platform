# Week 03 예제 - Claude 프롬프트 실전 예시

## 예제 1: 실제 Claude 대화 예시

### 포트폴리오 피드백 요청

**사용자 입력:**
```
당신은 10년 경력의 크리에이티브 디렉터입니다.
신입 그래픽 디자이너의 포트폴리오를 검토하는 역할입니다.

저는 컴퓨터그래픽디자인 전공 졸업 예정자입니다.
브랜드 디자이너 또는 콘텐츠 디자이너로 취업하고 싶습니다.

제 포트폴리오 구성입니다:
- 표지 (1페이지)
- 자기소개 (1페이지)
- 프로젝트 1: 카페 로고 리디자인 (4페이지)
- 프로젝트 2: 학교 포스터 디자인 (3페이지)
- 프로젝트 3: SNS 콘텐츠 제작 (2페이지)
- 기술/도구 (1페이지)
- 연락처 (1페이지)

총 13페이지 PDF입니다.

다음을 피드백해주세요:
1. 포트폴리오 구성의 강점과 약점
2. 브랜드 디자이너 취업에 더 잘 맞는 구성으로 개선 방안
3. 추가하면 취업에 유리한 프로젝트 유형
4. 페이지 수 적정성
```

**Claude 답변 예시:**
```
# 포트폴리오 피드백

## 강점
1. 13페이지로 적절한 분량입니다. (채용 담당자가 평균 3분 이내 검토)
2. 로고, 포스터, SNS로 다양한 분야를 보여주고 있습니다.
3. 자기소개 페이지 포함은 브랜딩 능력을 어필합니다.

## 개선점
1. **프로젝트 설명 부족**: 각 프로젝트에 "왜 이 디자인을 선택했는가"를 반드시 포함
2. **과정 부재**: 최종 결과물만 있고 스케치→시안→완성 과정이 없음
3. **타겟 직군 불명확**: 브랜드 디자이너 지망이라면 브랜드 관련 작업 강화 필요

## 브랜드 디자이너 맞춤 개선안
- 카페 로고 프로젝트를 "브랜드 아이덴티티 패키지"로 확장
  (로고 + 명함 + 패키지 + 어플리케이션)
- SNS 콘텐츠를 "소셜 미디어 브랜딩 가이드"로 발전
- 새로 추가할 프로젝트: 브랜드 매뉴얼 또는 패키지 디자인

## 추천 추가 프로젝트
1. 스타트업 브랜딩 프로젝트 (실제 또는 가상)
2. 제품 패키지 디자인
3. 기업 홍보물 시리즈

## 페이지 수 권고
현재 13페이지 → 목표 16~20페이지 (각 프로젝트 과정 추가)
```

## 예제 2: 단계별 프롬프트 개선 과정

### 1단계: 처음 프롬프트 (구체성 낮음)
```
자기소개서 써줘
```
결과: 너무 일반적인 내용

### 2단계: 개선된 프롬프트
```
컴퓨터그래픽디자인 전공자의 자기소개서 써줘
카카오 지원용으로
```
결과: 조금 나아졌지만 아직 부족

### 3단계: 완성된 프롬프트
```
당신은 취업 전문 컨설턴트입니다.

상황:
- 지원자: 컴퓨터그래픽디자인 전공 2026년 2월 졸업 예정
- 지원 기업: 카카오
- 지원 직군: UX 디자이너 (신입)
- 나의 경험: 카페 브랜딩 프로젝트, 학교 UI 리디자인, CGD 플랫폼 실습
- 나의 강점: 사용자 중심 사고, Adobe 툴 숙련, AI 도구 활용

자기소개서를 작성해주세요:
- 분량: 800자 내외
- 항목: 지원 동기 + 보유 역량 + 입사 후 포부
- 카카오의 "함께하는 즐거움" 가치관 반영
- 구체적인 수치와 경험 포함
- 10~20대가 쓴 것처럼 자연스럽게

추가로: 이 자기소개서에서 더 강화하면 좋은 부분 3가지도 알려주세요.
```
