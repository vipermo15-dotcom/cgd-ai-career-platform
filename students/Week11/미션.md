## 미션 목표
 실무 역량을 증명하는 결과물을 만듭니다.

---

## 필수 미션 (전부 완료 필수)

### 미션 A: 핵심 작업 완성
```
[ ]  관련 주요 결과물 1개 완성
[ ] Claude AI로 최소 3회 피드백 반영
[ ] 최종본 저장 (파일명 규칙: week11_이름_최종.확장자)
```

### 미션 B: 문서화
```
[ ] 이번 주 작업 과정 Markdown으로 정리
[ ] 배운 점, 어려웠던 점, 개선 방향 기술
[ ] GitHub에 업로드 완료
```

### 미션 C: 피드백 반영 보고서
```
[ ] Claude AI에게 완성된 결과물 평가 요청
[ ] 평가 결과 정리 (강점 + 개선점)
[ ] 개선사항 반영 후 최종 수정
[ ] 수정 전후 비교 파일 작성
```
