# Week 10 미션 - 자기소개서 작성

## 미션 개요
이번 주는 자기소개서 작성 주제로 실질적인 결과물을 만듭니다.

---

## 필수 미션

### 미션 1: 메인 작업
```
[ ] 자기소개서 작성 관련 주요 작업 수행
[ ] AI 도구 활용하여 품질 향상
[ ] 결과물 저장 및 정리
[ ] 캡처 또는 파일 제출
```

### 미션 2: AI 피드백 받기
```
[ ] Claude에게 이번 주 작업물 피드백 요청
[ ] 피드백 내용 정리 (최소 5항목)
[ ] 피드백 반영하여 1회 이상 수정
[ ] 수정 전/후 비교 자료 작성
```

### 미션 3: GitHub 업로드
```
[ ] 이번 주 작업물 GitHub에 push
[ ] 커밋 메시지 규칙 준수
[ ] 최소 3개 이상 커밋
[ ] README 업데이트
```

---

## 도전 미션
```
[ ] 이번 주 주제와 관련된 온라인 강의 1강 완수
[ ] 관련 Behance 작품 3개 분석 후 인사이트 정리
```
