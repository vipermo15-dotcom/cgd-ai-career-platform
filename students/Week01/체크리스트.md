## 수업 중 체크리스트

### Git 설치 및 설정
```
[ ] Git 설치 완료
      명령어: git --version
      결과: git version 2.x.x
      
[ ] 사용자 이름 설정
      명령어: git config --global user.name "이름"
      
[ ] 이메일 설정
      명령어: git config --global user.email "이메일"
      
[ ] 설정 확인
      명령어: git config --list
```

### 첫 번째 저장소
```
[ ] 로컬 폴더 생성 (my-portfolio 또는 portfolio-2026)
[ ] git init 실행
[ ] README.md 파일 생성
[ ] git add README.md 실행
[ ] git commit -m "첫 커밋" 실행
[ ] git log로 커밋 확인
```

### GitHub 연동
```
[ ] GitHub 계정 로그인
[ ] 새 저장소(repository) 생성
[ ] git remote add origin [URL] 실행
[ ] git push -u origin main 실행
[ ] GitHub에서 파일 확인
```

---

## 핵심 명령어 체크

아래 명령어를 모두 직접 실행해보고 체크하세요:

```
[ ] git init          - 저장소 초기화
[ ] git status        - 현재 상태 확인
[ ] git add .         - 모든 변경사항 추가
[ ] git commit -m ""  - 커밋 메시지와 함께 저장
[ ] git push          - GitHub에 업로드
[ ] git pull          - GitHub에서 최신 내용 내려받기
[ ] git log           - 커밋 기록 확인
[ ] git diff          - 변경사항 확인
```
