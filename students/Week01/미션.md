# Week 01 미션 - Git/GitHub 기초

## 미션 개요

이번 주 미션을 완료하면 GitHub에 나만의 첫 번째 포트폴리오 저장소가 생깁니다.
모든 미션은 캡처를 찍어 제출합니다.

---

## 필수 미션 (모두 완료해야 함)

### 미션 1: GitHub 계정 생성 및 프로필 설정
**목표**: 전문적인 GitHub 프로필 완성

```
완료 기준:
[ ] GitHub 계정 생성 (github.com)
[ ] 프로필 사진 업로드 (실물 또는 아바타)
[ ] 자기소개 (Bio) 작성: "컴퓨터그래픽디자인 전공 | 디자이너 지망생"
[ ] 제출: GitHub 프로필 URL
```

**예시 Bio**: `그래픽 디자이너 지망생 | Adobe Photoshop, Illustrator | CGD 양성과정`

---

### 미션 2: 포트폴리오 저장소 생성
**목표**: `portfolio-2026` 이름의 저장소 생성

```
완료 기준:
[ ] 저장소 이름: portfolio-2026
[ ] Public(공개) 설정
[ ] README.md 포함하여 생성
[ ] 제출: 저장소 URL (예: github.com/아이디/portfolio-2026)
```

---

### 미션 3: README.md 작성
**목표**: 자기소개가 담긴 README 작성

아래 템플릿을 참고하여 자신만의 README를 작성하세요.

```markdown
# 안녕하세요! 저는 [이름]입니다 👋

## 소개
- 🎨 컴퓨터그래픽디자인 전공
- 💼 목표 직군: [직군명]
- 📍 거주지: [지역]
- 📧 연락처: [이메일]

## 보유 스킬
- Adobe Photoshop
- Adobe Illustrator
- [기타 스킬]

## 현재 학습 중
- Claude AI 활용법
- GitHub 사용법
- [기타]

## 포트폴리오
준비 중입니다 🚀
```

```
완료 기준:
[ ] README.md에 자기소개 작성
[ ] GitHub에 push 완료
[ ] 저장소에서 README가 보이는 것 확인
[ ] 캡처 제출
```

---

### 미션 4: 첫 번째 커밋 기록 만들기
**목표**: 의미 있는 커밋 3개 이상 만들기

```bash
# 커밋 1: 초기 설정
git commit -m "init: 포트폴리오 저장소 초기화"

# 커밋 2: 자기소개 추가
git commit -m "docs: 자기소개 README 작성"

# 커밋 3: 목표 추가
git commit -m "docs: 취업 목표 및 스킬 목록 추가"
```

```
완료 기준:
[ ] git log로 3개 이상 커밋 확인
[ ] GitHub Insights에서 contribution 그래프 초록색 확인
[ ] 캡처 제출
```

## 도전 미션 (선택)

### 도전 1: .gitignore 파일 작성
```
# .gitignore 예시
.DS_Store
*.tmp
node_modules/
.env
```

### 도전 2: 브랜치 만들어보기
```bash
# 새 브랜치 생성
git branch feature/about-me

# 브랜치 이동
git checkout feature/about-me

# 작업 후 main에 병합
git checkout main
git merge feature/about-me
```

### 도전 3: GitHub Pages로 포트폴리오 웹페이지 만들기
1. Settings > Pages
2. Source: main 브랜치
3. index.html 파일 추가
4. 자신만의 URL 생성 확인
