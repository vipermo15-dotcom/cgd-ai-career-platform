## 실습 1: Firecrawl 계정 및 API Key

1. https://firecrawl.dev 접속
2. 회원가입 (GitHub 계정 연동 가능)
3. Dashboard → API Keys
4. "Create new key" 클릭
5. 키 복사 후 `.env` 파일에 저장

```bash
# .env 파일 (절대 GitHub에 올리면 안 됨!)
FIRECRAWL_API_KEY=fc-your-key-here
```

무료 플랜: 월 500 크레딧 (약 500회 스크래핑)

## 실습 3: 목표 기업 3곳 정보 수집

### 수집할 기업 선택 기준
- 내가 실제로 지원하고 싶은 기업
- 그래픽/UI/콘텐츠 디자이너 채용하는 기업
- 국내 기업 우선 (해외는 크레딧 소모 많음)

### 수집 스크립트

```python
# research_companies.py
from firecrawl import FirecrawlApp
from dotenv import load_dotenv
import os, json
from datetime import datetime

load_dotenv()
app = FirecrawlApp(api_key=os.getenv("FIRECRAWL_API_KEY"))

# 수집할 기업 목록 (본인 목표 기업으로 변경)
companies = [
    {"name": "카카오", "url": "https://careers.kakao.com"},
    {"name": "네이버", "url": "https://recruit.navercorp.com"},
    {"name": "쿠팡", "url": "https://www.coupang.jobs/kr"}
]

results = {}

for company in companies:
    print(f"수집 중: {company['name']}")
    
    try:
        data = app.scrape_url(
            url=company['url'],
            params={
                'formats': ['markdown'],
                'onlyMainContent': True,
                'waitFor': 2000
            }
        )
        
        results[company['name']] = {
            'url': company['url'],
            'collected_at': datetime.now().isoformat(),
            'content': data.get('markdown', '')[:3000]  # 처음 3000자
        }
        
        # 개별 파일로 저장
        os.makedirs("results", exist_ok=True)
        with open(f"results/{company['name']}.md", 'w', encoding='utf-8') as f:
            f.write(f"# {company['name']} 채용 정보\n\n")
            f.write(f"수집일: {datetime.now().strftime('%Y-%m-%d')}\n\n")
            f.write(data.get('markdown', ''))
        
        print(f"  → results/{company['name']}.md 저장 완료")
        
    except Exception as e:
        print(f"  → 오류: {e}")

print("\n모든 기업 정보 수집 완료!")
```

## 실습 4: Firecrawl Search로 채용공고 검색

```python
# search_jobs.py
from firecrawl import FirecrawlApp
import os
from dotenv import load_dotenv

load_dotenv()
app = FirecrawlApp(api_key=os.getenv("FIRECRAWL_API_KEY"))

# 채용공고 검색
results = app.search(
    query="그래픽 디자이너 신입 채용 2026",
    params={
        'limit': 10,
        'lang': 'ko'
    }
)

print(f"검색 결과: {len(results.get('data', []))}건\n")

for i, result in enumerate(results.get('data', []), 1):
    print(f"{i}. {result.get('title', '제목없음')}")
    print(f"   URL: {result.get('url', '')}")
    print(f"   설명: {result.get('description', '')[:100]}")
    print()
```
