## Firecrawl 설정 체크
```
[ ] firecrawl.dev 회원가입
[ ] API Key 발급 (Dashboard에서 확인)
[ ] pip install firecrawl-py 완료
[ ] pip install python-dotenv 완료
[ ] .env 파일 생성
[ ] .gitignore에 .env 추가 (필수!)
```

## 스크래핑 실습 체크
```
[ ] scrape.py 실행 성공
[ ] results/ 폴더에 .md 파일 생성 확인

[ ] 기업 3곳 수집 완료
```

## 보안 체크 (중요!)
