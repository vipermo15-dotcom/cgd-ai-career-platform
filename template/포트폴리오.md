## 4. 프로젝트 1~5 각 구성

### 프로젝트 표지 (1페이지)
```
[프로젝트 대표 이미지 - 전체 페이지]

프로젝트 01
─────────────────────
한국茶 브랜드 아이덴티티

Branding / Identity
2025년 10월
```

### 프로젝트 개요 (1페이지)
```
PROJECT OVERVIEW

클라이언트: 한국茶 (가상 브랜드)
기간: 2025년 9월 ~ 11월
분야: 브랜드 아이덴티티
도구: Illustrator, Photoshop, InDesign

────────────────────────────────

배경 및 목적
전통 차 문화를 MZ세대에게 어필하기 위한 
브랜드 아이덴티티 리디자인 프로젝트입니다.

핵심 과제
기존 브랜드의 정체성을 유지하면서
현대적 감성을 더하는 것이 핵심 과제였습니다.
```

### 과정 페이지 (1~2페이지)
```
DESIGN PROCESS

[리서치 무드보드 이미지]

리서치 → 컨셉 → 시안 → 최종

1단계: 리서치
경쟁 브랜드 분석과 타겟 설문을 통해 
"전통×현대"의 접점을 발견했습니다.

[컨셉 스케치 이미지]

2단계: 컨셉 개발
"한국의 자연에서 온 치유" 키워드를 
중심으로 세 가지 방향성을 제시했습니다.
```

## 파일 저장 규칙

```
파일명: 이름_직군_포트폴리오_연도.pdf
예시: 홍길동_브랜드디자이너_포트폴리오_2026.pdf

파일 크기: 20MB 이하 (이메일 제출용)
해상도: 150dpi 이상 (화면 기준 72dpi)
색상 모드: RGB (화면 제출) / CMYK (인쇄 제출)
페이지 크기: A4 (297×210mm)
```
