## 2. 채용공고 업데이트 방법

### 2.1 자동 업데이트 확인

```bash
# 최근 채용공고 수집 결과 확인
ls -lt data/jobs/ | head -10

# 가장 최근 파일 내용 확인
python3 -c "
import json
import os
import glob

files = sorted(glob.glob('data/jobs/*.json'), reverse=True)
if files:
    with open(files[0], 'r') as f:
        data = json.load(f)
    print(f'최신 파일: {files[0]}')
    print(f'수집 건수: {data.get(\"total_count\", 0)}건')
    print(f'수집일: {data.get(\"collected_at\", \"\")}')
"
```

### 2.2 수동 업데이트

자동화 실패 시 수동으로 채용공고를 수집합니다:

```bash
# 수동 수집 실행
python scripts/collect_jobs.py

# 특정 키워드로 수집
python scripts/collect_jobs.py --keyword "브랜드디자이너" --pages 3
```

## 3. 포트폴리오 예제 추가 방법

우수한 졸업생의 포트폴리오를 익명화하여 예제로 추가하는 절차:

### Step 1: 동의 확인
```
[ ] 학생에게 익명 예제 활용 동의 받기
[ ] 이름, 연락처 등 개인정보 제거 계획 확정
```

### Step 2: 파일 준비
```bash
# examples/ 폴더에 추가
cp student_portfolio.pdf examples/portfolio-samples/sample-brand-2026.pdf

# README 업데이트
echo "## 2026년 우수 포트폴리오 - 브랜드 디자인" >> examples/포트폴리오-예제.md
```

### Step 3: 설명 문서 작성
```markdown
## 예제: 브랜드 디자이너 포트폴리오 (2026년)

**수록 이유**: 브랜드 아이덴티티 작업의 완성도와 과정 설명이 우수
**강점**: 리서치 → 컨셉 → 최종 결과물까지 스토리텔링이 탁월
**배울 점**: 무드보드 활용과 폰트 선택 근거 명확히 기술

[파일 보기](portfolio-samples/sample-brand-2026.pdf)
```
