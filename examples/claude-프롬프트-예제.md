# Claude 대화 예제 - 포트폴리오 피드백 실전

## 개요
실제 Claude와의 대화를 통해 포트폴리오 피드백을 받는 과정 예시입니다.

---

## 대화 예제 1: 포트폴리오 목차 피드백

### 사용자 입력
```
당신은 10년 경력의 크리에이티브 디렉터입니다.
저는 CGD 전공 졸업 예정자로 브랜드 디자이너 취업을 준비 중입니다.

현재 포트폴리오 목차:
1. 표지 (1p)
2. 자기소개 (1p)
3. 로고 디자인 모음 (3p)
4. 포스터 작업 (2p)
5. SNS 콘텐츠 (2p)
6. 연락처 (1p)
총 10페이지

피드백 주세요:
1. 목차 구성의 강점과 약점
2. 브랜드 디자이너에 특화된 개선안
3. 추가하면 좋은 섹션
```

### Claude 답변
```
# 포트폴리오 목차 피드백

## 강점

**1. 분량이 간결합니다**
10페이지는 처음 제출 시 채용 담당자가 부담없이 볼 수 있는 분량입니다.
특히 첫 접촉에서 PDF를 이메일로 보낼 때 효과적입니다.

**2. 다양한 분야를 보여줍니다**
로고 + 포스터 + SNS 콘텐츠로 세 가지 분야를 커버한 점이 좋습니다.

---

## 약점 (개선 필수)

**1. "로고 디자인 모음"이 문제입니다**
브랜드 디자이너 포트폴리오에서 가장 큰 실수 중 하나가 로고만 나열하는 것입니다.
채용 담당자는 "이 디자이너가 왜 이 로고를 만들었는가"를 보고 싶어합니다.
단순 나열은 실력보다 작업 수만 보여줍니다.

**2. 브랜드 아이덴티티 '시스템'이 없습니다**
브랜드 디자이너에게 가장 중요한 것은 로고 1개가 아니라
로고 + 색상 + 폰트 + 어플리케이션이 하나의 시스템으로 작동하는 것을 보여주는 것입니다.

**3. 포스터와 SNS 콘텐츠는 브랜드 디자이너의 주력 업무가 아닙니다**
이 두 섹션이 있는 것은 나쁘지 않지만, 브랜드 디자이너로 어필하려면
브랜딩 작업이 포트폴리오의 70% 이상을 차지해야 합니다.

---

## 브랜드 디자이너 특화 개선안

### 수정된 목차 제안

```
01. 표지 (1p)
02. 자기소개 + 철학 (2p)
03. Project 1: [브랜드명] 아이덴티티 (5p)
    - 브리프 + 리서치
    - 컨셉 개발 과정
    - 로고 + 컬러 + 타이포 시스템
    - 명함/패키지/디지털 어플리케이션
04. Project 2: [브랜드명] 리브랜딩 (4p)
05. Project 3: [편집/패키지 등] (3p) 
06. Skills (1p)
07. Contact (1p)
총 17~18페이지
```

### 핵심 변경사항
- "로고 모음" → "브랜드 아이덴티티 시스템" (스토리가 있는 1개)
- 각 프로젝트에 제작 과정 반드시 포함
- 포스터/SNS는 독립 섹션보다 브랜딩 프로젝트의 어플리케이션으로 포함

---

## 추가 추천 섹션

**1. Design Process (과정 공개 페이지)**
- 스케치 → 시안 → 수정 → 완성
- "이 디자이너는 어떻게 생각하는가"를 보여줌

**2. 연구/인사이트 노트**
- 색상 선택 이유
- 폰트 선택 기준
- 차별화 포인트 설명

**예상 완성도 향상**: 현재 60점 → 수정 후 85점 예상

추가 질문 있으면 언제든지 말씀해주세요!
```

## 대화 예제 2: 자기소개서 피드백 후 수정

### Round 1 (초안)
```
사용자: 이 자기소개서 피드백 해주세요.

"안녕하세요, 저는 컴퓨터그래픽디자인을 전공한 홍길동입니다.
디자인에 열정을 가지고 있으며 최선을 다해 업무에 임하겠습니다.
귀사에서 성장하고 싶습니다."

Claude: 솔직하게 말씀드리면, 이 문장들은 채용 담당자가 하루에 50번 읽는 표현들입니다.

문제점:
1. "열정", "최선", "성장" - 모든 지원자가 쓰는 단어
2. 구체적인 경험이나 근거가 없음
3. 이 회사를 왜 선택했는지 없음
4. 내가 어떤 디자이너인지 전혀 알 수 없음

수정해볼게요:
"3년간의 브랜드 아이덴티티 프로젝트 경험이 저를 [기업명]으로 이끌었습니다.
[특정 프로젝트]에서 [구체적 성과]를 달성했고,
[기업명]의 [구체적 제품/서비스]에서 그 경험을 더 발전시키고 싶습니다."

이 방향으로 수정해보세요. 그리고 구체적인 경험이 있다면 알려주시면 
더 잘 도와드릴 수 있어요.
```
