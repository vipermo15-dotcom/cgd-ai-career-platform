# 포트폴리오 구성 예시

## 예시 개요
브랜드 디자이너 취업을 준비하는 가상의 학생 "김예진"의 포트폴리오 구성 예시입니다.
이 예시를 참고하여 본인만의 포트폴리오를 구성하세요.

---

## 김예진 Brand Portfolio 2026

### 기본 정보
- 이름: 김예진 (가상)
- 직군: 브랜드 디자이너
- 총 페이지: 22페이지
- PDF 크기: 14.5MB
- 제작 도구: Adobe InDesign, Illustrator, Photoshop

---

## 목차 구성

| 페이지 | 섹션 | 내용 |
|--------|------|------|
| 1 | Cover | 이름, 직군, 한 줄 철학 |
| 2~3 | About Me | 자기소개 + 스킬 인포그래픽 |
| 4~8 | Project 01 | 유기농 식품 브랜드 "GROW" 아이덴티티 |
| 9~13 | Project 02 | 로컬 카페 "달빛" 리브랜딩 |
| 14~17 | Project 03 | 인천시 청년 문화축제 포스터 시리즈 |
| 18~19 | Skills | Adobe CC + AI 도구 활용 |
| 20~21 | Process | 제작 과정 공개 (스케치 → 완성) |
| 22 | Contact | 연락처 + QR코드 |

### Project 01: GROW 브랜드 아이덴티티 (4~8페이지)

```
Page 4: 프로젝트 표지
  - 대형 로고 이미지
  - "GROW - Organic Brand Identity"
  - 기간: 2025.09~11, 도구: Illustrator, Photoshop

Page 5: 배경 + 리서치
  - 클라이언트 브리프 요약
  - 경쟁사 분석 (무드보드)
  - 타겟 페르소나

Page 6: 컨셉 개발
  - 키워드: 자연, 신뢰, 순수
  - 스케치 3가지 방향성
  - 색상 팔레트 선정 과정

Page 7: 로고 개발
  - 로고 시안 3개
  - 선정된 로고 설명 (왜 이것인가)
  - 사용 규정 (색상별, 크기별)

Page 8: 어플리케이션
  - 명함, 패키지, 웹 배너에 적용된 모습
  - 실제 제품 목업 (3D)
```

## 포트폴리오 제작 팁 (이 예시에서 배울 점)

1. **스토리텔링**: 각 프로젝트에 "왜 이 디자인인가"를 명확히 설명
2. **과정 공개**: 완성작만이 아니라 스케치와 시안 단계도 포함
3. **목업 활용**: 실제 제품처럼 보이는 3D 목업으로 설득력 향상
4. **일관성**: 전체 포트폴리오가 하나의 브랜드처럼 통일감 있음
5. **분량 적절**: 22페이지, 채용 담당자가 5분 내 볼 수 있는 분량
