"한국디자인그룹"은 교육 목적으로 만든 가상 기업입니다.

### 수집된 원본 데이터 (발췌)

```markdown
# 한국디자인그룹 채용 안내

## 회사 소개
한국디자인그룹(KDG)은 2010년 설립된 종합 디자인 컨설팅 기업입니다.
브랜딩, 패키지, UX/UI를 중심으로 국내외 300여 개 기업의 프로젝트를 수행했습니다.
임직원 150명, 연간 매출 200억원의 중견 디자인 에이전시입니다.

## 채용 직군

### 그래픽 디자이너 (신입/경력)
**업무 내용**
- 브랜드 아이덴티티 디자인
- 편집 디자인 (브로셔, 카탈로그)
- 포장 디자인
- 판촉물 및 옥외광고물 제작

**지원 자격**
- 관련 전공자 우대 (비전공자도 지원 가능)
- Adobe CC 활용 능력 필수 (Photoshop, Illustrator, InDesign)
- 포트폴리오 필수 제출

**우대 사항**
- 영어 또는 중국어 가능자
- 인쇄/제판 지식 보유자
- AI 디자인 도구 활용 가능자
- 공모전 수상 경력자

**근무 조건**
- 위치: 서울시 마포구 합정동
- 근무시간: 9:00~18:00 (유연근무 가능)
- 급여: 회사 내규에 따름 (신입 2,800~3,200만원 예상)
- 복리후생: 4대 보험, 연차, 교육비 지원, 워크숍

## 인재상
KDG는 "창조하고 소통하고 성장하는 디자이너"를 찾습니다.
- 창조: 새로운 시각으로 문제를 해결하는 창의적 사고
- 소통: 클라이언트와 팀원과 원활히 소통하는 능력
- 성장: 끊임없이 배우고 발전하려는 자세
```

## 3. 최종 보고서

### 기업 분석 종합

**강점**
- 설립 16년의 안정된 에이전시 (신입 안전망)
- 다양한 분야 경험 가능 (브랜딩, 패키지, 편집)
- 중견 규모로 책임감 있는 역할 부여
- AI 도구 활용자 우대 (CGD 과정 이수자 유리)

**약점**
- 연봉이 IT 기업 대비 낮음
- 에이전시 특성상 야근 가능성
- 스타트업/IT 비해 창의적 자율성 낮을 수 있음

**나의 준비 상태 평가**

| 필요 역량 | 내 수준 | 상태 |
|----------|---------|------|
| Photoshop 중급 | ★★★★☆ | 충족 |
| Illustrator 중급 | ★★★☆☆ | 보완 필요 |
| InDesign | ★★☆☆☆ | 보완 필요 |
| 브랜딩 포트폴리오 | 있음 | 충족 |
| 인쇄 지식 | ★★☆☆☆ | 보완 필요 |

**지원 전략**
1. 포트폴리오에 브랜딩 작업 앞에 배치
2. 자소서에 "소통"과 "성장" 키워드 자연스럽게 녹이기
3. InDesign 기초 실습 1주일 집중 후 지원
4. 지원 이메일에 Behance 링크 + PDF 첨부

**최종 결정**: 지원 (B 우선순위, A 우선순위 기업 지원 후 보완용)
