- 선호 포트폴리오 스타일: 캠페인 기획력 + 디지털 실행력 중심
- 강조할 역량: SNS 광고 콘텐츠 제작 능력, AI 툴 활용 효율화 사례

**취업처 2: 대홍기획**
- 업종·규모: 종합광고대행사 (롯데그룹 계열, 대기업)

- 강조할 역량: 브랜드 가이드라인 적용 능력, 인쇄·디지털 통합 작업물

**취업처 3: 29CM / 무신사**
- 업종·규모: 국내 패션·라이프스타일 이커머스 플랫폼 (중대형)
- 선호 포트폴리오 스타일: 감성·브랜딩 중심, 트렌디하고 세련된 비주얼
- 강조할 역량: SNS 콘텐츠, 배너·상세페이지 제작 경험, 숏폼

**취업처 4: 카카오 스타일 (지그재그)**
- 업종·규모: 패션 버티컬 커머스 플랫폼 (카카오 계열)
- 선호 포트폴리오 스타일: 데이터 감각 + 트렌디한 디지털 콘텐츠
- 강조할 역량: 숏폼 기획·편집, AI 활용 콘텐츠 제작 속도·효율

## 작업물별 설명 가이드

### 작업물 1: SNS 광고 콘텐츠 (카드뉴스·배너)
**한 줄 소개**: 플랫폼별 타깃과 목적에 맞게 기획·제작한 디지털 광고 콘텐츠
**강조할 것**: 기획 의도 설명 + AI 툴로 시안 탐색한 과정 스크린샷 포함
**설명 순서**: 타깃·목적 → 기획 의도 → 디자인 결정 근거 → AI 활용 방식
**피할 것**: "예쁘게 만들었다"는 표현. 성과 수치 없으면 과장 금지

### 작업물 2: 숏폼 영상 편집본
**한 줄 소개**: 기획부터 편집까지 단독으로 제작한 숏폼 영상 콘텐츠
**강조할 것**: Premiere + AE + AI 툴 조합 제작 과정. 스크립트→편집본 전 과정
**설명 순서**: 기획 배경 → 스크립트·콘티 → 편집 포인트 → AI 활용 구간
**피할 것**: 영상 길이만 강조. 완성도보다 과정과 의도가 핵심

### 작업물 3: 브랜드 아이덴티티 작업물
**한 줄 소개**: 브랜드 컨셉부터 로고·컬러·적용 사례까지 일관되게 구성한 브랜드 패키지
**강조할 것**: 톤앤매너 일관성, 브랜드 스토리와 디자인의 연결 근거
**설명 순서**: 브랜드 컨셉 → 로고 도출 과정 → 컬러·타이포 → 적용 사례
**피할 것**: 로고만 단독으로 보여주는 것. 반드시 적용 목업 포함

### 작업물 4: 굿즈 패키지 디자인
**한 줄 소개**: 브랜드 세계관을 실물 굿즈로 확장한 패키지 디자인
**강조할 것**: 인쇄 적용 고려 여부, 브랜드와의 연계성. 라이프스타일 감성 강조
**설명 순서**: 브랜드 연결 배경 → 소재·형태 선택 이유 → 최종 결과물
**피할 것**: 굿즈 자체만 보여주는 것. 브랜드 맥락 없으면 설득력 약함

### 작업물 5: 편집디자인 (리플렛·매거진 등)
**한 줄 소개**: InDesign을 활용해 정보 위계와 레이아웃 완성도를 높인 편집물
**강조할 것**: 그리드 시스템 활용, 가독성과 시각적 흐름 설계
**설명 순서**: 제작 목적 → 레이아웃 설계 근거 → 타이포·컬러 선택 → 최종본
**피할 것**: 완성본 이미지만 나열. 과정과 판단 근거를 반드시 포함
