**직무 2: 브랜드·광고 콘텐츠 디자이너**

- 추가 역량: 브랜드 가이드라인 작성 경험

**직무 3: SNS·숏폼 콘텐츠 크리에이터 (AI 연계)**
- 이유: Premiere·After Effects + 4종 AI 툴 보유는 현재 시장 희소성 높음
- 추가 역량: 플랫폼별 포맷 이해 (릴스·틱톡·유튜브 쇼츠)

## 준비 로드맵 (잡지사·출판사 위주)

### 단기 (1개월)

**포트폴리오 기반 작업**
- InDesign으로 잡지 레이아웃 샘플 1종 제작 (A4 8페이지 이상, 가상 매거진 가능)
- 북커버 디자인 2~3종 제작 (장르별: 문학·실용·에세이)
- 기존 편집디자인 작업물 정리 및 PDF 포트폴리오 1차 완성

**기술 보완**
- 인쇄 제작 사양 학습: 블리드(도련) 3mm, CMYK 색상 모드, 해상도 300dpi 확인
- InDesign 단축키·단락 스타일·마스터 페이지 숙달

---

### 중기 (3개월)

**취업처별 맞춤 준비**

- **디자인하우스**: 기존 잡지 레이아웃 분석 후 동일 스타일 모작 1종 → 포트폴리오 추가
- **매거진B**: 관심 브랜드 1개 선정 → 브랜드 분석 에세이(2장) + 해당 브랜드 가상 매거진 레이아웃 제작
- **민음사·문학동네**: 실제 출판사 신간 북커버 리디자인 2종 + 내지 샘플 4페이지

**지원 활동**
- 사람인·원티드·잡코리아 편집디자이너·북디자이너 공고 북마크 및 주 1회 확인
- 포트폴리오 PDF 최종본 완성 (10페이지 이내, 표지 포함)
- 자기소개서 초안 작성 (취업처별 1종씩)
