# 포트폴리오 구성 가이드 — 민지

## 취업처 특성 분석

취업처 1: 매거진·출판사 (디자인하우스, 시공사 등)
- 업종·규모: 중대형 출판사 / 매거진 편집부
- 선호 포트폴리오 스타일: 깔끔·정보전달 중심 / 그리드 완성도 중시
- 강조할 역량: InDesign 레이아웃 완성도, 타이포그래피 감각, 잡지 형식 편집 경험

취업처 2: 콘텐츠 디자인 회사 (크릭앤리버코리아, 디스트릭트, 스튜디오좋 등)
- 업종·규모: 중소형 콘텐츠 스튜디오
- 선호 포트폴리오 스타일: 트렌디한 비주얼 + AI 활용 제작 효율 / 콘셉트 완성도
- 강조할 역량: SNS·광고콘텐츠 제작 경험, AI 툴 활용 프로세스, 빠른 시안 제작력

취업처 3: 브랜딩 에이전시 (슬로워크, Plus X, Studio fnt 등)
- 업종·규모: 중소형 에이전시
- 선호 포트폴리오 스타일: 콘셉트 기획력 + 비주얼 완성도 / 다양한 매체 대응력
- 강조할 역량: AI 활용 시안 제작 속도, 브랜드 아이덴티티 기획·적용 능력

취업처 4: 국공립 미술관·박물관 (국립현대미술관, 서울시립미술관 등)
- 업종·규모: 공공기관 / 디자인팀 소규모
- 선호 포트폴리오 스타일: 기관 BI 준수 + 문화적 맥락 이해 / 절제된 고급감
- 강조할 역량: 전시 포스터·사인물 제작, 동양화 전공 배경, 문화예술교육사 2급

취업처 5: 문화재단·예술단체 (서울문화재단, 국립극단 등)
- 업종·규모: 공공·준공공기관
- 선호 포트폴리오 스타일: 공익성·가독성 중심 / 기관 정체성 반영
- 강조할 역량: 홍보물·편집 디자인, 공모전 수상 이력, 문화예술 현장 경험

## 작업물별 설명 가이드

## 작업물 1: 편집디자인 레이아웃 (잡지 형식)

한 줄 소개: 그리드와 타이포그래피로 콘텐츠의 흐름을 설계한 잡지 형식 편집 디자인 작업
강조할 것: InDesign 레이아웃 완성도 / 여백·서체 선택 근거 / 페이지 간 흐름 (잡지 형식 기준)
설명 순서: 기획 의도 → 그리드 구조 설명 → 서체·색상 선택 이유 → 완성 결과
피할 것: "예쁘게 만들었다"는 표현 / 툴 나열 위주 설명 / 페이지 수 자랑

---

## 작업물 2: 브랜드 아이덴티티 + 굿즈

한 줄 소개: 브랜드 콘셉트 기획부터 굿즈 적용까지 일관된 아이덴티티를 구현한 작업
강조할 것: 콘셉트 → BI 설계 → 굿즈 확장 흐름 / Midjourney 무드보드 활용 과정
설명 순서: 브랜드 배경·타깃 → 키워드 도출 → 로고·컬러 → 굿즈 적용 → AI 활용 지점
피할 것: 굿즈 사진만 나열 / 콘셉트 근거 없는 디자인 / AI 활용 사실 숨기기

---

## 작업물 3: SNS 콘텐츠 시리즈 (3~5컷)

한 줄 소개: 플랫폼 특성에 맞춰 시리즈 통일감과 전달력을 동시에 잡은 콘텐츠 디자인
강조할 것: 시리즈 간 톤·무드 일관성 / ChatGPT 카피 + Midjourney 이미지 협업 과정
설명 순서: 타깃·플랫폼 설정 → 콘셉트 → AI 활용 과정 → 후보정 → 최종 게시 포맷
피할 것: 컷 수 자랑 / AI가 다 했다는 인상 / 플랫폼 포맷 무시한 비율

---

## 작업물 4: AI 활용 광고콘텐츠 (제작과정 포함)

한 줄 소개: 프롬프트 설계부터 후보정까지, AI와 협업하는 디자인 프로세스를 보여주는 작업
강조할 것: 프롬프트 → Midjourney 결과 → Photoshop 후보정 → 완성본 단계 시각화
설명 순서: 광고 목적·타깃 → 프롬프트 설계 근거 → 중간 결과물 → 보정 과정 → 최종본
피할 것: 완성본만 제출 (과정 생략 금지) / 프롬프트 공개 꺼리는 태도 / AI 의존도 과장

---

## 작업물 5: 전시 포스터 or 사인물 그래픽

한 줄 소개: 전시 주제와 작품 세계를 시각 언어로 풀어낸 문화예술 특화 그래픽 작업
강조할 것: 동양화 전공 배경에서 나온 미감 / 기관 BI 준수 + 절제된 고급감 / 문화적 맥락 이해
설명 순서: 전시 주제 이해 → 시각 콘셉트 도출 → 색상·서체·이미지 선택 근거 → 완성
피할 것: 화려한 효과 남발 / 기관 성격과 맞지 않는 톤 / 작품 이미지 무단 편집 느낌
