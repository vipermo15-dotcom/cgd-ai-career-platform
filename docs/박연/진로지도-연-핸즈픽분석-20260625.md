## 5. 이력서·자기소개서 분석

### 강점
- 회화·만화 전공으로 감성 해석력 + 콘셉트 기획력 보유
- 광주 동구청 자서전 삽화 → 클라이언트 납품 실무 경험
- 선문대 홍보 웹툰 → 목적 기반 이미지 제작 경험
- 굿즈 소비자 경험 + 소장 욕구 분석 언급 (자소서) → 차별화 포인트
- 성격 단점 솔직하게 + 개선 노력 서술 → 신뢰감

### 보완 필요 사항
- **경력 1년 이상 요건**: 디자인 실무 경력 미충족 → 교육원 과정 + 프로젝트를 실무 준하는 경험으로 기술 필요
- **Adobe 툴 수준**: Illustrator 역량이 포트폴리오로 직접 증명 필요
- **패키지·굿즈 디자인 작업물**: 이력서상 굿즈 제작 이력 없음 → 교육원 과정 작업물로 구성
- **자소서 마무리**: "핸즈픽의 시즌 컬렉션처럼~" 회사 제품 언급으로 구체화 권장

## 6. 포트폴리오 구성 전략 (핸즈픽 맞춤)

> 핵심 키워드: **기획 + 패키지 + 시즌감**
> 연의 회화·만화 배경을 굿즈 기획력으로 재해석하는 것이 핵심

### 작업물 1 ★ 최우선
**시즌 굿즈 패키지 디자인 (목업 포함)**
- 콘셉트 예시: "전시 작품에서 영감받은 아트 굿즈 패키지"
- 오리지널 일러스트 → 스티커·엽서·포장 박스 세트 구성
- 패키지에 재단선·도수 표기 (인쇄 실무 이해 어필)
- 목업 이미지로 실제 상품처럼 완성
- 기획 배경 → 콘셉트 → 제작 → 결과 흐름 1페이지로 정리
- **설명 순서**: 기획 배경 → 디자인 콘셉트 → 제작 과정 → 목업 결과물
- **피할 것**: 목업 없이 평면 시안만 제출 / 인쇄 규격 미표기

### 작업물 2
**캐릭터 기반 굿즈 라인업**
- 만화애니메이션 전공 강점 → 오리지널 캐릭터 시트 제작
- 캐릭터 정면·측면·표정·컬러팔레트 구성
- 해당 캐릭터 적용 굿즈 2~3종 목업 (노트·파우치·뱃지 등)
- "이 캐릭터를 왜 만들었는가" 기획 의도 한 줄 필수
- **설명 순서**: 캐릭터 기획 의도 → 시트 구성 → 굿즈 적용 사례 → 확장 가능성
- **피할 것**: 캐릭터 시트 없이 완성 일러스트만 제출 / 세계관 설명 없는 그림 나열

### 작업물 3
**클라이언트 대응 시안 — 자서전 삽화 or 홍보 웹툰 재구성**
- 광주 동구청 자서전 삽화: "클라이언트 요청 → 시안 → 납품" 흐름으로 재편집
- 선문대 홍보 웹툰: 목적·대상·결과 중심으로 1페이지 정리
- 단순 작품 나열 ✕ → 과정과 의도 중심으로 재구성 ✅
- **설명 순서**: 브랜드/클라이언트 설정 → 요청 사항 → 시안 제작 → 납품 결과
- **피할 것**: AI 활용 사실만 언급하고 과정 미첨부 / 카드 수 과다 (5~7장 이내)

### 작업물 4 (선택)
**트렌드 반응 팬시 아이템**
- 최근 문구·팬시 트렌드 분석 → 직접 제작한 아이템 1종
- "왜 이 트렌드를 선택했는가" 짧은 리서치 메모 첨부
