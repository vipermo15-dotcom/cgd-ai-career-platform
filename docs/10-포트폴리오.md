## 1. 포트폴리오란?

포트폴리오(Portfolio)는 본인의 작업물을 모아 역량을 보여주는 작품집입니다.

### 디자이너에게 포트폴리오가 중요한 이유

- 디자인 면접의 70%는 포트폴리오 기반으로 진행
- 말보다 작업물이 실력을 더 잘 증명
- 첫 인상을 결정하는 핵심 도구
- 지원자를 차별화하는 가장 강력한 수단

### 채용 담당자가 포트폴리오에서 보는 것

| 평가 항목 | 내용 |
|-----------|------|
| 기술 역량 | 툴 숙련도, 결과물 퀄리티 |
| 사고력 | 디자인 의도, 문제 해결 과정 |
| 다양성 | 여러 분야 작업 경험 |
| 성장 가능성 | 최근 작업의 발전 정도 |
| 커뮤니케이션 | 작업 설명의 명확성 |

## 2. 포트폴리오 구성 원칙

### 원칙 1: 질 > 양

```
❌ 30개 작업물 (평균 품질)
✅ 10개 작업물 (높은 품질)
```

채용 담당자는 포트폴리오를 평균 5-10분 이내 검토합니다.  
최고의 작업물 8-12개를 신중하게 선별하세요.

### 원칙 2: 첫 작업물이 가장 중요

포트폴리오의 첫 번째 작업물이 전체 인상을 결정합니다.  
**가장 자신 있는 작업물을 첫 번째에 배치**하세요.

### 원칙 3: 스토리가 있는 구성

단순히 결과물만 보여주는 것이 아니라,  
**제작 과정과 사고 과정**을 함께 보여주세요.

```
좋은 작업물 소개 구조:
1. 문제 (어떤 과제였나?)
2. 접근 (어떻게 생각했나?)
3. 제작 과정 (어떻게 만들었나?)
4. 결과물 (최종 산출물)
5. 배운 점 (이 작업으로 성장한 것)
```

### 원칙 4: 목표 직종에 맞게 커스터마이징

지원하는 직종에 따라 포트폴리오 구성을 달리하세요.

| 직종 | 강조할 작업 유형 |
|------|----------------|
| 그래픽 디자이너 | 브랜딩, 편집, 타이포그래피 |
| UI/UX 디자이너 | 앱 UI, 웹 디자인, 사용자 리서치 |
| 영상/모션 디자이너 | 모션 그래픽, 영상 편집 |
| 광고 디자이너 | 캠페인, 광고물, 기획 |
| 편집 디자이너 | 책, 잡지, 브로슈어 |

### 원칙 5: 일관된 비주얼 아이덴티티

포트폴리오 자체도 하나의 디자인 작업물입니다.  
- 일관된 레이아웃 사용
- 통일된 폰트 2-3종
- 브랜드 컬러 1-2가지
- 통일된 마진과 여백

## 3. 디자인 포트폴리오 vs 일반 포트폴리오

### 디자인 포트폴리오의 특징

```
일반 이력서:
텍스트 중심 → 경력 나열

디자인 포트폴리오:
비주얼 중심 → 실력 직접 증명
```

### 유형별 포트폴리오 특성

**1. PDF 포트폴리오**
- 장점: 오프라인 공유 가능, 레이아웃 고정
- 단점: 파일 크기 관리 필요
- 권장 크기: A4 또는 16:9 비율, PDF 10MB 이하

**2. 웹 포트폴리오 (Behance)**
- 장점: 무료, 글로벌 노출, 피드백 수집
- 단점: 인터넷 필요
- URL: `behance.net/사용자명`

**3. Notion 포트폴리오**
- 장점: 쉬운 업데이트, 링크 공유
- 단점: 커스터마이징 제한
- URL: `notion.so/사용자명`

**4. GitHub 포트폴리오**
- 장점: 개발자/기술 분야 선호
- 단점: 비디자이너에게 생소
- URL: `github.com/사용자명`

**5. 개인 웹사이트**
- 장점: 완전한 커스터마이징
- 단점: 제작 시간, 유지비

## 4. 필수 포함 항목 체크리스트

### 포트폴리오 기본 항목

**커버 (표지)**
- [ ] 이름
- [ ] 직종 (예: Graphic Designer)
- [ ] 연락처 (이메일)
- [ ] 날짜 (20XX년)

**소개 페이지**
- [ ] 자기소개 (3-5문장)
- [ ] 보유 기술 목록 (Adobe CC, Figma 등)
- [ ] 원하는 직종 명시
- [ ] SNS/포트폴리오 사이트 링크

**작업물 섹션 (각 작업물 필수)**
- [ ] 프로젝트 제목
- [ ] 카테고리 (브랜딩/편집/UI 등)
- [ ] 클라이언트 또는 목적
- [ ] 사용 툴
- [ ] 작업 기간
- [ ] 제작 과정 설명 (최소 2-3문장)
- [ ] 최종 결과물 이미지 (고해상도)
- [ ] 목업 이미지 (실제 사용 환경 시뮬레이션)

**마감 페이지**
- [ ] 감사 인사
- [ ] 연락처
- [ ] "포트폴리오 전문은 별도 첨부" 안내 (선택)

### 포트폴리오 품질 체크리스트

**이미지 품질**
- [ ] 최종 이미지 해상도 300dpi 이상 (인쇄용)
- [ ] 화면용 72dpi 이상
- [ ] 배경이 깔끔한 목업 이미지 사용
- [ ] 이미지 픽셀 깨짐 없음

**내용 품질**
- [ ] 오탈자 없음
- [ ] 일관된 폰트 사용 (2-3종)
- [ ] 충분한 여백 (답답하지 않은 레이아웃)
- [ ] 페이지 번호 표기

## 5. AI 활용 포트폴리오 제작 방법

### Claude로 작업 설명 작성하기

```
내 포트폴리오의 [프로젝트명] 작업에 대한 설명을 써줘.

작업 정보:
- 프로젝트명: 카페 '브루잉' 브랜드 아이덴티티
- 목적: 신규 카페 오픈을 위한 브랜드 아이덴티티 개발
- 내가 한 것: 로고 디자인, 명함, 컵 홀더, 패키지
- 사용 툴: Adobe Illustrator, Photoshop
- 기간: 2주

형식:
- 영어로 작성 (Behance용)
- 2단락 (프로젝트 소개 + 디자인 결정 이유)
- 각 단락 3-4문장
- 전문적이면서 따뜻한 어조
```

### Gemini로 트렌드 리서치

```
2024년 그래픽 디자인 포트폴리오 트렌드를 검색해줘.
특히 한국 취업 시장에서 선호하는 포트폴리오 스타일을 알려줘.
신입 디자이너가 피해야 할 실수도 알려줘.
```

### ChatGPT로 영문 번역

```
아래 포트폴리오 작업 설명을 자연스러운 영어로 번역해줘.
Behance에 게시할 것이므로 전문적인 디자인 용어를 사용해줘.

한국어 원문:
[번역할 내용]
```

### AI로 목업 프롬프트 작성 (Midjourney)

```
내 포트폴리오 목업 이미지를 Midjourney로 만들 프롬프트를 작성해줘.

목업 유형: 커피컵 패키지 디자인
분위기: 고급스럽고 모던한 카페 분위기
배경: 나무 테이블, 자연광
색감: 따뜻한 베이지 톤

영어 Midjourney 프롬프트로 작성해줘.
```

## 6. 플랫폼별 포트폴리오 구성

### Behance 포트폴리오

**프로젝트 설정**:
1. Behance 가입 (Adobe ID 사용)
2. "New Project" 클릭
3. 제목: `Brand Identity - Brewing Coffee`
4. 툴: Illustrator, Photoshop
5. Creative Fields: Graphic Design, Branding

**프로젝트 구성 (섹션 순서)**:

```
① 커버 이미지 (2560×1440px)
② 프로젝트 소개 텍스트 (영어)
③ 초기 스케치/아이디어 과정
④ 컬러 팔레트, 폰트 선택
⑤ 로고 변형 (가로형, 세로형, 심볼)
⑥ 실제 적용 목업
   - 명함
   - 컵 홀더
   - 패키지
⑦ 마무리 이미지
```

**Behance SEO 팁**:
- 제목에 키워드 포함 (Branding, Logo Design, Identity 등)
- 태그 최대 10개 추가
- 프로젝트 설명에 사용 툴 명시

### Notion 포트폴리오

**기본 구조**:
```
[홈페이지]
├── About Me (자기소개)
├── Skills (기술 목록)
├── Portfolio (작업물)
│   ├── 브랜딩
│   ├── 편집 디자인
│   └── UI/UX
└── Contact (연락처)
```

**Notion 포트폴리오 공유**:
1. 페이지 우상단 "Share" 클릭
2. "Share to web" 켜기
3. URL 복사 → 이력서에 첨부

## 7. 포트폴리오 발표 방법

### 3분 포트폴리오 발표 구조

```
[서론 - 30초]
"저는 브랜드 아이덴티티 디자인에 관심이 많습니다.
오늘 3가지 작업물을 중심으로 포트폴리오를 설명하겠습니다."

[작업 1 - 60초]
"첫 번째는 [프로젝트명]입니다.
[문제 상황] → [내 접근 방식] → [결과]
이 작업에서 특히 [배운 점]을 얻었습니다."

[작업 2 - 60초]
(동일 구조)

[마무리 - 30초]
"제 포트폴리오는 아직 부족한 부분이 있지만,
이 회사에서 [구체적 기여 방향]을 통해 
더 성장하고 싶습니다. 감사합니다."
```

### 면접관의 질문에 대비하기

자주 나오는 포트폴리오 질문:

**Q1: "이 작업에서 가장 어려웠던 점은?"**
```
준비할 내용:
- 구체적인 기술적 어려움 한 가지
- 어떻게 해결했는지 과정
- 그 경험에서 배운 것
```

**Q2: "다시 한다면 어떻게 바꾸겠나요?"**
```
준비할 내용:
- 솔직한 개선점 한 가지
- 지금이라면 다르게 할 구체적인 방법
- 성장을 보여주는 답변
```

**Q3: "이 색상을 선택한 이유는?"**
```
준비할 내용:
- 타겟 고객과 브랜드 특성 연결
- 컬러 심리학적 근거
- 경쟁사와의 차별화
```
