# 포트폴리오 가이드 — 김S × 공공기관·재단 홍보 디자이너

---

## 작업물 1: 공공기관 스타일 카드뉴스

**한 줄 소개**: 정부 디자인 가이드라인을 준수한 정보 전달형 카드뉴스
**강조할 것**: 텍스트 위계 구조, 여백 활용, 색상 절제 — 공공기관 담당자가 "우리 스타일을 아는 디자이너"로 인식하게 만드는 포인트
**설명 순서**: 제작 배경(어떤 정보를 전달하려 했나) → 레이아웃 결정 근거 → 최종 결과물
**피할 것**: 과도한 그래픽 효과, 유행 폰트 남용, 브랜드 컬러 임의 변경

---

## 작업물 2: 재단·비영리 홍보 리플릿

**한 줄 소개**: InDesign으로 제작한 다단 편집 기반 비영리 홍보 인쇄물
**강조할 것**: 다단 그리드 구성력, 따뜻한 톤앤매너, 인쇄 출력 고려한 여백·해상도 설정
**설명 순서**: 대상 독자 설정 → 레이아웃 구조 설명 → 인쇄 사양(도련·재단선 등) → 결과물
**피할 것**: "예쁘게 만들었다"는 표현 — 대신 "정보 접근성을 높이기 위해 ~했다"로 설명

---

## 작업물 3: 홍보 영상 (30초~1분)

**한 줄 소개**: Premiere Pro + After Effects로 제작한 기관 홍보용 숏폼 영상
**강조할 것**: 컷 편집 흐름, 자막 가독성, 모션그래픽 활용 — 1인이 기획~완성까지 담당했음을 어필
**설명 순서**: 기획 의도(무엇을 알리려 했나) → 편집 구조 → 사용 툴·효과 → 최종 영상
**피할 것**: 배경음악 저작권 미확인 음원 사용, 너무 긴 러닝타임 (1분 초과 시 편집 버전 별도 준비)

---

## 작업물 4: AI 활용 콘텐츠 제작물

**한 줄 소개**: Midjourney·Claude를 활용한 기획~제작 전 과정이 담긴 작업물
**강조할 것**: AI 툴을 단순 사용이 아닌 "기획 도구"로 활용했다는 점 — 프롬프트 설계 과정, 결과물 선별 기준, 후보정 과정까지 보여주기
**설명 순서**: 프롬프트 기획 → AI 생성 결과 비교 → 선택 근거 → 후보정(Photoshop 등) → 최종 결과물
**피할 것**: AI가 만든 것을 그대로 제출했다는 인상 — 반드시 "내 판단과 편집이 들어간 결과물"임을 강조

## 작업물 5: 연간보고서·백서 편집 샘플

**한 줄 소개**: InDesign 기반 긴 호흡의 다단 편집 — 목차·색인·페이지 구성까지 완성한 샘플
**강조할 것**: 마스터 페이지 활용, 단락 스타일 통일, 목차 자동화 — 실무 편집 능력을 증명하는 포인트
**설명 순서**: 편집 구조 설계(페이지 수·섹션 구성) → 마스터 페이지·스타일 설명 → 완성 페이지 발췌
**피할 것**: 텍스트가 임시 더미(Lorem Ipsum)인 채로 제출 — 실제 주제가 있는 내용으로 채울 것
