## 1. 취업지원 시스템 개요

### 플랫폼 구성 요소

```
CGD AI Career Platform 취업지원 시스템
│
├── 기업 DB (Firecrawl로 수집된 기업 정보)
│   ├── 기업분석DB/ 폴더
│   └── 채용공고DB/ 폴더
│
├── 지원 현황 관리 (GitHub Projects)
│   ├── 칸반 보드
│   └── Issues 트래킹
│
├── 자료 라이브러리 (이 저장소)
│   ├── template/ 문서 템플릿
│   ├── prompt/ AI 프롬프트
│   └── workflow/ 업무 흐름
│
└── AI 어시스턴트 (Claude/Gemini/ChatGPT)
    ├── 자기소개서 작성·첨삭
    ├── 기업분석 요약
    └── 면접 준비
```

### 취업 준비 전체 타임라인

```
수료 6개월 전:
  → 포트폴리오 기획 시작
  → GitHub 정리 시작
  → 관심 기업 리스트 작성

수료 3개월 전:
  → 포트폴리오 80% 완성
  → 자기소개서 초안 작성
  → 기업 분석 본격 시작

수료 1개월 전:
  → 포트폴리오 최종 완성
  → 자기소개서 확정
  → 지원 시작

수료 후:
  → 집중 지원
  → 면접 준비
  → 피드백 수집 및 개선
```

## 2. 채용 플랫폼 활용

### 주요 채용 플랫폼

**사람인 (saramin.co.kr)**
- 가장 많은 공고 수
- 대기업부터 중소기업까지
- 프리미엄 이력서 기능

**잡코리아 (jobkorea.co.kr)**
- 공기업, 대기업 공채 강점
- 연봉 정보 제공
- 이직 준비자에게 유리

**원티드 (wanted.co.kr)**
- IT, 스타트업 강점
- 추천 기반 채용
- 합격 시 추천인에게 보상

**링크드인 (linkedin.com)**
- 외국계 기업, 글로벌 기업
- 헤드헌터 접근 가능
- 네트워킹 도구

**디자인 특화 플랫폼**
- **크몽 (kmong.com)**: 프리랜서 디자인 일감
- **탈잉 (taling.me)**: 재능 공유
- **Behance Jobs**: 디자인 전문 채용

### 효율적인 플랫폼 활용법

**알림 설정**:
```
사람인: 직군 "그래픽 디자이너" + 지역 "서울" + 경력 "신입" 설정 후 알림
잡코리아: 관심 직종 등록 → 신규 공고 이메일 수신
원티드: 관심 포지션 저장 → 유사 공고 추천 받기
```

**Firecrawl로 자동 수집**:
```python
# 매일 새 채용공고 자동 수집
job_search_urls = [
    "https://www.saramin.co.kr/zf_user/jobs/list/job-category?cat_kewd=85&panel_type=none&search_optional_item=y",
    "https://www.wanted.co.kr/jobs?category=design",
]
```

## 3. 지원 현황 트래킹

### GitHub Projects 칸반 보드 활용

**카드 작성 예시**:
```markdown
## 카카오 - 브랜드 디자이너

### 기본 정보
- 공고 URL: https://...
- 마감일: 2024-XX-XX
- 예상 연봉: X,XXX만원
- 규모: 대기업

### 준비 현황
- [x] 기업 분석 완료
- [x] 자기소개서 초안
- [ ] 포트폴리오 커스터마이징
- [ ] 최종 제출

### 면접 일정
- 1차: 미정
- 2차: 미정

### 메모
- 강조 포인트: 카카오 서비스 사용 경험, 브랜드 이해
- 주의사항: 포트폴리오에 모바일 UI 작업 추가 필요
```

### 스프레드시트 병행 활용

Google Sheets로 추가 트래킹:

```
| 회사명 | 직무 | 마감일 | 지원일 | 서류결과 | 면접일 | 최종결과 | 비고 |
|-------|-----|------|------|---------|------|---------|-----|
| 카카오 | 브랜드디자이너 | 24.01.31 | 24.01.25 | 통과 | 24.02.10 | 대기 | |
```

## 4. AI 기반 지원서 최적화

### 회사별 자기소개서 버전 관리

폴더 구조:
```
취업지원/
├── 기업분석/
│   ├── 카카오.md
│   ├── 네이버.md
│   └── 배민.md
├── 자기소개서/
│   ├── 기본형.md
│   ├── 카카오_버전.md
│   └── 네이버_버전.md
└── 지원결과/
    ├── 결과기록.md
    └── 면접후기.md
```

### 커스터마이징 프롬프트

```
내 기본 자기소개서를 [회사명] 맞춤형으로 수정해줘.

기본 자기소개서:
[기본형 자기소개서]

[회사명] 특성:
[기업 분석 내용]

수정 방향:
- 지원동기에 [회사명] 특정 서비스/제품 언급 추가
- [회사명]의 핵심 가치 키워드 자연스럽게 포함
- [회사명] 디자인 방향성과 내 역량 연결
- 내용은 그대로, 표현과 강조점만 조정
```

## 5. 네트워킹 전략

### 오프라인 네트워킹

**디자인 커뮤니티 참여**:
- 디자이너스 파티 (월간 디자이너 네트워킹)
- GDSC (Google Developer Student Club)
- 로컬 디자인 밋업 (Meetup.com 검색)

**포트폴리오 리뷰 행사**:
- Behance Reviews
- 디자인 학교 포트폴리오 데이
- 기업 Open House

### 온라인 네트워킹

**링크드인 활용**:
```
1. 프로필 완성도 100%
2. 헤드라인 최적화: "그래픽 디자이너 | CGD 과정 수료"
3. 포트폴리오 섹션에 Behance 링크
4. 관심 기업 팔로우
5. 채용 담당자 연결 요청 (메시지 포함)
```

**연결 요청 메시지 예시**:
```
안녕하세요 [이름]님,
저는 서울시기술교육원 컴퓨터그래픽디자인과 교육생 [내 이름]입니다.
[회사명]의 디자인 문화에 관심이 많아 연결 요청드립니다.
혹시 시간이 되신다면 커피챗으로 경험을 나눠주실 수 있을까요?
```

## 6. 합격 후 준비사항

### 합격 통보 받은 후 체크리스트

**즉시 해야 할 것**:
- [ ] 합격 수락 의사 전달 (합격 메일 수신 후 24시간 이내)
- [ ] 입사 일자 확정
- [ ] 다른 전형 중인 기업에 취소 연락
- [ ] 합격 기록 (GitHub Issues 업데이트)

**입사 전까지**:
- [ ] 근로계약서 내용 확인 (연봉, 업무, 복리후생)
- [ ] 건강검진 (요구 시)
- [ ] 필요 서류 준비 (주민등록등본 등)
- [ ] 입사 전 온보딩 자료 확인

**입사 첫 주**:
- [ ] 팀원 인사
- [ ] 업무 툴 세팅 (Mac/Windows, Adobe CC 등)
- [ ] 사내 규정 숙지
- [ ] 첫 주 학습 계획 수립
