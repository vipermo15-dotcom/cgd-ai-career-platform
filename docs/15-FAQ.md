## 도구 관련 FAQ

**Q: Claude, ChatGPT, Gemini 중 어떤 것을 써야 하나요?**

A: 용도에 따라 다르게 사용하는 것을 권장합니다.
- **Claude**: 자기소개서 작성, 기업분석, 긴 문서 처리 → 가장 추천
- **Gemini**: 최신 채용 정보 검색, 기업 최신 뉴스, 이미지 분석
- **ChatGPT**: 영어 번역, 영문 이력서, 창의적 글쓰기

하나만 선택해야 한다면 **Claude**를 추천합니다.

---

**Q: 모든 도구가 유료인가요?**

A: 기본 기능은 모두 무료입니다.
- Claude: [claude.ai](https://claude.ai) 무료 계정
- GitHub: 무료 계정
- VSCode: 완전 무료
- Gemini: Google 계정으로 무료
- ChatGPT: [chatgpt.com](https://chatgpt.com) 무료 계정

Firecrawl은 교수자 계정을 통해 제공됩니다.

---

**Q: 컴퓨터 사양이 낮아도 사용할 수 있나요?**

A: 네, 대부분의 도구가 웹 기반이므로 인터넷 연결만 있으면 됩니다.
- Claude, Gemini, ChatGPT: 웹브라우저에서 실행 (사양 무관)
- GitHub: 웹브라우저에서 모두 가능
- VSCode: 가벼운 편 (RAM 4GB 이상 권장)

## Claude 관련 FAQ

**Q: Claude 무료 버전의 사용량 제한이 있나요?**

A: 있습니다. 시간당 메시지 수가 제한되어 있습니다.
- 대화가 길어지면 "사용량 한도에 도달했습니다" 메시지가 나타남
- 1-3시간 후 다시 사용 가능
- 취업 준비 일상 사용에는 충분한 수준
- 많이 사용하고 싶다면 Pro 버전 고려 ($20/월)

---

**Q: Claude에 개인정보를 입력해도 안전한가요?**

A: 다음 사항을 지키면 안전합니다.
- 주민등록번호, 계좌번호, 비밀번호 등 민감 정보는 절대 입력하지 않기
- 이름, 연락처 정도는 자기소개서 작성에 필요한 수준에서 사용 가능
- Claude는 대화 내용을 AI 학습에 사용할 수 있으므로 민감한 정보는 제외

---

**Q: Claude가 생성한 자기소개서를 그대로 제출해도 되나요?**

A: 절대 금지입니다.
- AI 생성 글은 채용 담당자가 구별할 수 있습니다
- 자신의 실제 경험이 없으면 면접에서 검증 불가
- 채용 윤리 위반으로 합격 취소 가능

올바른 사용법:
1. Claude로 초안 작성
2. 본인 경험과 언어로 완전히 재작성
3. Claude에게 첨삭 요청
4. 최종 확인 후 제출

---

**Q: Claude 대화가 갑자기 끊기면 어떻게 하나요?**

A: 다음 방법으로 대화를 이어갑니다.
```
"위의 [주제] 대화를 이어서 계속해줘.
마지막으로 [요청한 내용]을 요청했었는데
거기서부터 계속 진행해줘."
```
또는 새 대화를 시작하고 이전 맥락을 요약하여 제공하세요.

## GitHub 관련 FAQ

**Q: GitHub는 꼭 사용해야 하나요?**

A: 강력히 권장합니다.
- 포트폴리오 호스팅 (무료)
- 학습 이력 시각화 (잔디)
- 취업 시장에서 GitHub 링크 제출이 일반화됨
- 디자이너도 GitHub 계정이 있으면 차별화됨

---

**Q: GitHub에 포트폴리오 이미지를 올리면 용량 문제가 있나요?**

A: 용량 주의가 필요합니다.
- 파일당 100MB 제한
- 저장소 전체 1GB 권장 (5GB 이상 시 경고)
- 고해상도 이미지는 압축 후 업로드
- 큰 이미지는 Google Drive 또는 Behance에 올리고 링크 공유 권장

---

**Q: 실수로 잘못된 파일을 Push 했어요.**

A: 걱정하지 마세요. 수정하는 방법이 있습니다.

방법 1 (단순 수정):
```bash
# 파일 수정 후
git add 파일명
git commit -m "fix: 잘못된 내용 수정"
git push origin main
```

방법 2 (삭제):
```bash
git rm 파일명
git commit -m "remove: 잘못된 파일 삭제"
git push origin main
```

API 키 같은 민감한 정보를 올렸다면 즉시 API 키를 재발급하고 강사에게 문의하세요.

## 포트폴리오 관련 FAQ

**Q: 작업물이 부족한데 어떻게 포트폴리오를 만들어요?**

A: 다음 방법으로 작업물을 늘릴 수 있습니다.
1. **리디자인 프로젝트**: 실제 기업의 디자인을 개인 프로젝트로 재해석
   - "카카오 앱 리디자인", "OO 브랜드 리브랜딩"
2. **자체 프로젝트**: 가상의 클라이언트를 설정하고 작업
   - "친구 카페 로고 제작", "학교 행사 포스터"
3. **수업 과제 고도화**: 수업 과제를 더 완성도 높게 다듬기
4. **공모전 참가**: 결과와 관계없이 제출 작업물을 포트폴리오에 포함

---

**Q: 포트폴리오 PDF는 어떻게 만드나요?**

A: 여러 도구를 활용할 수 있습니다.

**Adobe InDesign (권장)**:
- 전문적인 레이아웃 작업
- 포트폴리오 제작에 최적

**Canva (무료 대안)**:
- 간편한 드래그 앤 드롭
- 무료 템플릿 제공
- [canva.com](https://canva.com)

**PowerPoint/Keynote**:
- 익숙한 인터페이스
- PDF로 내보내기 가능

---

**Q: 포트폴리오를 영어로 만들어야 하나요?**

A: 지원 기업에 따라 다릅니다.
- 국내 기업: 한국어 OK, 영어 병기하면 더 좋음
- 외국계 기업: 영어 필수
- Behance: 영어로 작성하면 글로벌 노출

한국어 포트폴리오를 만들고, 영어 설명을 추가하는 방식을 권장합니다.

## 취업 준비 관련 FAQ

**Q: 비전공자인데 디자이너로 취업 가능한가요?**

A: 충분히 가능합니다.
- 채용 시 학력보다 포트폴리오를 봅니다
- 비전공자의 강점: 다른 분야 경험, 새로운 관점
- CGD 과정 수료 = 실무 능력 증명
- 적극적인 포트폴리오 제작과 꾸준한 공모전 참가가 중요

---

**Q: 취업까지 보통 얼마나 걸리나요?**

A: 개인차가 크지만 일반적으로:
- 빠른 경우: 수료 전후 1-2개월
- 평균: 수료 후 3-6개월
- 긴 경우: 6-12개월

포트폴리오 완성도와 지원 기업 수가 가장 중요합니다.

---

**Q: 연봉이 얼마나 되나요?**

A: 신입 그래픽 디자이너 기준 (2024년):
- 소규모 기업: 2,400-2,800만원
- 중견기업: 2,800-3,500만원
- 대기업: 3,500만원 이상

잡플래닛(glassdoor.co.kr)에서 기업별 연봉 정보를 확인하세요.

---

**Q: 자격증이 필요한가요?**

A: 필수는 아니지만 도움이 됩니다.
- **컴퓨터그래픽스운용기능사**: 기술 능력 증명
- **웹디자인기능사**: 웹 분야 지원 시 유리
- **GTQ (그래픽기술자격)**: 포토샵 능력 증명

자격증보다 포트폴리오가 더 중요합니다.

## Firecrawl 관련 FAQ

**Q: Firecrawl은 모든 사이트에서 작동하나요?**

A: 대부분 작동하지만 예외가 있습니다.
- 로그인이 필요한 페이지: 접근 불가
- 봇 차단 기술이 있는 사이트: 작동 불안정
- 동적 JavaScript 페이지: 느리지만 대부분 작동

---

**Q: 채용공고를 스크래핑하면 불법인가요?**

A: 개인 학습 목적은 일반적으로 허용됩니다.
- 개인 학습·분석 목적의 소규모 스크래핑은 허용
- 상업적 사용 또는 대규모 수집은 사이트 이용약관 확인 필요
- 교육 목적으로 사용하는 경우 문제없음
