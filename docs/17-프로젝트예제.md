## 1. 포트폴리오 프로젝트 예제

### 예제: 카페 브랜드 아이덴티티 프로젝트

**프로젝트 설명 (한국어)**

```markdown
# Brew & Co. 카페 브랜드 아이덴티티

## 프로젝트 개요
- **클라이언트**: 가상 클라이언트 (자체 프로젝트)
- **기간**: 2024년 1월 (2주)
- **역할**: 브랜드 전략, 로고 디자인, 아이덴티티 전체

## 브랜드 스토리
"천천히, 제대로"를 모토로 하는 스페셜티 커피 전문점 브랜드.
바쁜 서울 직장인들이 잠깐의 여유를 즐길 수 있는 공간.

## 디자인 방향
- 컬러: 따뜻한 브라운(#8B4513) + 크림(#FFF8DC) + 그린(#2E8B57)
- 폰트: Playfair Display (영문) + 나눔명조 (한글)
- 컨셉: 클래식하면서도 현대적, 고급스러우면서도 따뜻한

## 제작 과정
1. **리서치**: 국내외 스페셜티 커피 브랜드 20개 분석
2. **스케치**: 로고 아이디어 30개 초안 작성
3. **디지털 제작**: 상위 5개 선별 후 Illustrator로 제작
4. **피드백**: 지인 10명 대상 선호도 조사
5. **최종 완성**: 1순위 로고로 전체 아이덴티티 완성

## 산출물
- 로고 (컬러/흑백/역배경 3종)
- 명함 앞뒤
- 테이크아웃 컵 홀더
- 종이봉투
- SNS 프로필 이미지

## 배운 점
로고 하나가 전체 브랜드 경험을 결정한다는 것을 배웠습니다.
처음 만든 로고와 최종 로고의 차이를 비교하면서
"단순함"이 얼마나 어려운 것인지 깨달았습니다.
```

**Behance용 영어 설명**:

```
Brew & Co. is a specialty coffee brand designed for Seoul's busy professionals 
who seek a moment of calm in their hectic days. The visual identity balances 
classic craftsmanship with modern minimalism.

The warm brown palette evokes the richness of well-crafted coffee, 
while the cream accents bring lightness and approachability. 
The custom logotype combines elegance with readability across all touchpoints 
from business cards to takeaway packaging.
```

## 2. 기업분석 보고서 예제

### 예제: 카카오 기업분석 보고서

```markdown
# 카카오 기업분석 보고서

**분석일**: 2024년 1월 15일  
**작성자**: 홍길동  
**출처**: kakao.com, 카카오 블로그, 뉴스

---

## 1. 기업 개요

### 한 줄 요약
> 카카오는 카카오톡을 기반으로 메신저, 커머스, 콘텐츠, 금융을 연결하는 국내 최대 모바일 플랫폼 기업입니다.

### 주요 사업
- **카카오톡**: 월 4,500만 명 사용자의 국민 메신저
- **카카오쇼핑**: 커머스 서비스
- **카카오페이**: 간편결제·금융 서비스
- **카카오T**: 교통·모빌리티 서비스
- **카카오엔터**: 웹툰, 음악, 영상 콘텐츠

---

## 2. 기업 문화

### 미션
"더 나은 세상을 위한 기술과 사람"

### 핵심 가치
1. **카카오답게**: 카카오의 정체성을 지키며 일하기
2. **함께**: 동료와 협력하며 성장
3. **성장**: 개인과 조직이 함께 성장

### 일하는 방식
- 유연한 근무 환경 (재택·사무실 혼합)
- 수평적 문화 (호칭: OO님)
- 데이터 기반 의사결정

---

## 3. 디자인팀 분석

### 카카오 디자인 조직
카카오 디자인센터에서 전사 디자인을 담당합니다.

### 디자인 특징
- **컬러**: 노란색(#FFE400)이 메인, 전 서비스에 일관 적용
- **타이포**: KakaoSans 커스텀 폰트 개발하여 브랜드 일관성
- **UX**: 친근하고 직관적인 인터페이스 강조
- **아이콘**: 단순하고 인식하기 쉬운 형태

---

## 4. 면접 준비 포인트

### 이 회사에서 강조해야 할 것
1. **카카오 서비스 실사용 경험**: 카카오톡, 카카오페이 등 직접 사용 경험
2. **브랜드 일관성 이해**: 노란색 아이덴티티의 일관된 적용 이해
3. **사용자 중심 디자인**: 4,500만 명 사용자를 위한 UX 관점
4. **협업 능력**: 대규모 조직에서의 협업

### 예상 면접 질문

**Q1: 카카오 서비스 중 디자인이 마음에 드는 것과 개선할 점은?**
준비 방향:
- 카카오톡, 카카오페이, 카카오맵 중 하나 선택
- 구체적으로 "무엇이 좋고 왜 좋은지" (디자인 관점)
- 개선 제안: 실제로 써본 경험에서 불편했던 점 → 개선 방향

**Q2: 수천만 명이 쓰는 서비스의 디자이너가 되었을 때 어떻게 접근하겠나요?**
준비 방향:
- 사용자 다양성 고려 (노인, 어린이, 다양한 환경)
- 작은 변경도 큰 영향 → 데이터 기반 의사결정
- 접근성(Accessibility) 고려

## 5. 지원 전략

### 포트폴리오 강조 포인트
- 모바일 UI/UX 경험
- 일관된 브랜드 시스템 작업
- 대규모 사용자를 고려한 설계 사고

### 자기소개서 키워드
사용자경험, 일관성, 접근성, 모바일퍼스트, 협업, 데이터기반

### 역질문
- "카카오 디자인팀에서 신입이 처음 맡게 되는 업무는 어떤 것인가요?"
- "카카오 디자인 스타일 가이드가 어떻게 관리되나요?"
```

## 3. 자기소개서 완성 예제

### 예제: 그래픽 디자이너 지원

**성장과정 (600자)**

```
고등학교 2학년 겨울방학, 학교 소식지 레이아웃을 처음 맡았습니다.
워드프로세서만 알던 저는 사진과 글을 예쁘게 배치하려다가
꼬박 이틀을 소비하며 구글을 뒤졌습니다.
결국 어설프게 완성한 소식지를 교실에 붙였을 때,
"이번엔 좀 예쁘다"는 친구들의 말에 멈추지 못하고 계속 고쳤습니다.
그게 디자인의 시작이었습니다.

서울시기술교육원 컴퓨터그래픽디자인과에 입학하면서
디자인이 단순한 꾸밈이 아닌 소통의 언어라는 것을 배웠습니다.
완성도보다 "왜 이렇게 디자인했는가"를 먼저 묻는 교육 방식은
저의 디자인 사고를 근본부터 바꿔놓았습니다.

특히 브랜딩 수업에서 6번의 피드백을 통해 로고 하나를 완성한 경험은,
좋은 디자이너란 처음부터 잘 만드는 사람이 아니라
피드백을 두려워하지 않고 꾸준히 발전하는 사람임을 가르쳐주었습니다.
지금의 저는 빠르지 않아도 제대로 성장하고 싶은 디자이너입니다.
```

## 5. AI 활용 프로세스 예제

### 취업 준비 1주차 AI 활용 기록

**월요일**: 기업 분석

```
사용 도구: Claude + Firecrawl
작업 내용:
1. Firecrawl로 카카오 홈페이지 수집 (15분)
2. Claude에 분석 요청 → 기업분석 보고서 초안 (10분)
3. 수동으로 내용 보완 (30분)
4. GitHub 저장소에 커밋 (5분)
총 소요 시간: 60분
```

**화요일**: 채용공고 분석

```
사용 도구: Claude
작업 내용:
1. 카카오 채용공고 복사
2. Claude에 키워드 추출 요청
3. 키워드를 자기소개서 개요에 반영
총 소요 시간: 30분
```

**수요일**: 자기소개서 작성

```
사용 도구: Claude
작업 내용:
1. 내 경험 정리 (20분)
2. Claude에 성장과정 초안 요청 (5분)
3. 초안을 내 언어로 수정 (45분)
4. Claude에 첨삭 요청 (5분)
5. 피드백 반영 수정 (20분)
총 소요 시간: 95분
```

**목요일**: 모의 면접

```
사용 도구: Claude
작업 내용:
1. 모의 면접 설정 (5분)
2. 10개 질문 답변 연습 (45분)
3. Claude 피드백 반영 (20분)
4. 면접 답변 문서화 (15분)
총 소요 시간: 85분
```
