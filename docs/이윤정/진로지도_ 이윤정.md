## 5. 포트폴리오 구성 전략

### 전체 컨셉

> **"공간을 직접 만져본 디자이너가 그래픽까지 합니다"**  
> VMD 실무 경력 → 그래픽 역량 추가의 스토리로 전체 포트폴리오 구성

### 작업물 구성 (4개 권장)

#### 작업물 1: VMD 경험 재구성 ★ 핵심 차별화
- **유형**: 공간 비주얼 케이스스터디
- **내용**: 홈쇼핑/VMD 현장 디스플레이 → 기획 의도·배치 근거·결과 포함 재구성
- **강조**: 고객 동선·공간 맥락 고려한 비주얼 구성 능력
- **주의**: 전 직장 사진 사용 시 회사 허가 여부 확인 필요

#### 작업물 2: 브랜드 아이덴티티 + 공간 적용 목업
- **유형**: 브랜드디자인
- **내용**: 브랜드 아이덴티티 → 매장·팝업 환경 목업으로 확장
- **강조**: 로고·컬러 시스템 → 사이니지·배너 적용까지 일관성
- **팁**: Mockup 사이트 활용, 대형 출력물·매장 간판 시뮬레이션 추가

#### 작업물 3: 출력물 연계 그래픽 (인쇄 이해도 증명)
- **유형**: 편집디자인 / 광고 콘텐츠
- **내용**: 재단선·도련·300dpi 등 인쇄 사양 적용한 작업물
- **강조**: 출력물 제작 및 인쇄 프로세스 이해
- **없다면**: 기존 작업물을 인쇄 사양에 맞게 재작업 후 제출

#### 작업물 4: AI 활용 시즌 비주얼 시안
- **유형**: AI 활용 광고 콘텐츠
- **내용**: Midjourney로 팝업/시즌 캠페인 시안 생성 → Photoshop 매장 목업 완성
- **강조**: 빠른 시안 제작 능력 + 공간 적용 감각
- **서술 순서**: 시안 기획 의도 → AI 프롬프트 → 후편집 → 적용 목업

### 작업물 서술 구조 (무신사 공고 요구사항 반영)

```
① 문제/배경: 어떤 목적으로 만들었나
② 내 역할:   무엇을 혼자 했나 (기여도 명확히)
③ 해결 과정: 어떤 판단을 했나
④ 결과:      최종 결과물 + 출력·적용 고려 사항
```
