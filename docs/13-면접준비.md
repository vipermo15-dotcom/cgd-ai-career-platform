## 1. 면접 종류 이해

### 1차 면접 (실무 면접)

- **면접관**: 디자인팀 팀장 또는 선임 디자이너
- **시간**: 30-60분
- **주요 내용**:
  - 포트폴리오 발표 및 질의응답
  - 직무 능력 평가
  - 기술 역량 확인
  - 팀 문화 적합성 파악

**평가 기준**:
- 포트폴리오 설명력 (이유와 과정 설명)
- 디자인에 대한 관점
- 기술적 역량 수준
- 열정과 성장 가능성

### 2차 면접 (임원/HR 면접)

- **면접관**: 임원진, HR 담당자
- **시간**: 30-45분
- **주요 내용**:
  - 인성 및 가치관 검증
  - 회사 문화 적합성
  - 장기 근무 의향
  - 처우 협상

**평가 기준**:
- 회사에 대한 이해도
- 커리어 비전의 명확성
- 조직 적합성
- 커뮤니케이션 능력

### 실기 테스트 (일부 기업)

일부 기업은 면접 전후로 실기 테스트를 실시합니다:

| 테스트 유형 | 내용 | 소요 시간 |
|------------|------|-----------|
| 디자인 과제 | 특정 주제의 포스터/SNS 카드뉴스 제작 | 2-4시간 |
| 툴 실력 테스트 | 포토샵/일러스트 실시간 작업 | 30분-1시간 |
| 포트폴리오 발표 | PPT 또는 PDF 발표 | 10-15분 |

## 2. 자주 나오는 질문 30가지

### 인성 관련 질문 (10가지)

**Q1: 자기소개를 해주세요.**
```
답변 팁:
- 1-2분 내로 간결하게
- 이름, 교육 배경, 보유 기술, 지원 이유 순서
- 마지막에 "잘 부탁드립니다" 로 마무리
```

**Q2: 왜 디자이너가 되고 싶으셨나요?**
```
답변 팁:
- 구체적인 계기와 에피소드
- "어릴 때부터 미술을 좋아했어서" 는 부족함
- 디자인으로 해결하고 싶은 문제가 있는 사람으로 어필
```

**Q3: 가장 힘들었던 경험과 극복 방법을 말해주세요.**
```
답변 팁:
- STAR 기법 (상황-과제-행동-결과)
- 결과뿐 아니라 배운 점에 집중
- 포기하지 않는 끈기 어필
```

**Q4: 자신의 장단점을 말해주세요.**
```
답변 팁:
장점: 업무와 연관된 구체적인 장점 + 에피소드
단점: 실제 단점 + 개선 노력 (치명적 단점은 피함)
```

**Q5: 5년 후 본인의 모습은?**
```
답변 팁:
- 이 회사 내에서의 성장 그림
- 너무 높은 목표 (CEO 등)는 피함
- 전문성 심화 or 리더십 방향 중 선택
```

**Q6: 왜 우리 회사에 지원하셨나요?**
```
답변 팁:
- 반드시 회사 특성 언급 (경쟁사와 구분되는 것)
- 회사의 특정 제품/서비스/문화 언급
- "유명해서" "좋아 보여서" 금지
```

**Q7: 이직 이유가 있나요? (경력 있는 경우)**
```
답변 팁:
- 이전 회사 험담 절대 금지
- 성장과 도전을 위한 긍정적 이유
- 이 회사에서 원하는 것과 연결
```

**Q8: 팀워크 경험을 말해주세요.**
```
답변 팁:
- 구체적인 협업 프로젝트 사례
- 본인의 역할과 기여
- 갈등 상황과 해결 과정 (있다면)
```

**Q9: 마감 기한을 맞추지 못한 경험이 있나요?**
```
답변 팁:
- 솔직하게 인정 (없다고 하면 신뢰성 저하)
- 원인과 대처 방법
- 이후 개선한 습관
```

**Q10: 궁금한 점이 있으신가요? (역질문)**
```
좋은 역질문 예시:
- "디자인팀의 하루 일과가 어떻게 되나요?"
- "입사 후 처음 3개월 동안 주로 어떤 업무를 하게 되나요?"
- "팀에서 가장 필요로 하는 역량이 무엇인가요?"
- "디자인 결정 권한이 어느 정도 있는지 궁금합니다"
```

### 직무 관련 질문 (15가지)

**Q11: 포트폴리오에서 가장 자신 있는 작업물은?**
```
답변 팁:
- 결과물이 좋은 것 + 과정을 잘 설명할 수 있는 것
- 문제 → 접근 → 결과 → 배운 점 구조
- 수치나 피드백 포함 (가능하면)
```

**Q12: 이 작업에서 가장 어려웠던 점은?**
```
답변 팁:
- 기술적 어려움 or 클라이언트 소통 어려움
- 어떻게 해결했는지가 핵심
- 성장 경험으로 마무리
```

**Q13: 디자인 영감은 어디서 얻으시나요?**
```
좋은 답변:
- Behance, Dribbble, Pinterest (구체적 플랫폼)
- 일상 생활에서의 관찰
- 타 분야에서의 크로스 인사이트 (건축, 패션 등)
```

**Q14: 최근 인상 깊었던 디자인이 있다면?**
```
답변 팁:
- 미리 2-3개 준비
- 왜 좋았는지 디자인적 관점에서 설명
- 이 회사 제품/경쟁사 제품을 언급하면 더 좋음
```

**Q15: 비전공자인데 어떻게 디자인을 배웠나요? (해당 시)**
```
답변 팁:
- 독학 과정의 구체적 방법
- 어떤 자료, 강의를 활용했는지
- 비전공자의 장점 어필 (다른 분야의 시각)
```

**Q16: Adobe CC 중 가장 잘 쓰는 툴은?**
```
답변 팁:
- 단순히 "포토샵"이 아닌 구체적 기능 언급
- 해당 툴로 만든 작업물과 연결
- 배우고 싶은 툴도 함께 언급 (학습 의지)
```

**Q17: Figma 사용 경험이 있나요?**
```
답변 팁:
- 있다면: 컴포넌트, 프로토타이핑 등 구체적 경험
- 없다면: 배우는 중 또는 배울 의지 표현 (구체적으로)
```

**Q18: 클라이언트(or 상사)와 의견 충돌이 있었다면?**
```
답변 팁:
- 의견 차이를 어떻게 해소했는지
- 설득 vs 수용의 판단 기준
- 결과와 배운 점
```

**Q19: 디자인 피드백을 어떻게 받아들이나요?**
```
답변 팁:
- 피드백을 성장의 기회로 보는 긍정적 태도
- 피드백 반영 과정 구체적으로
- 좋은 피드백과 부당한 피드백을 구별하는 기준
```

**Q20: 트렌드를 어떻게 따라가나요?**
```
답변 팁:
- 구체적인 플랫폼, 커뮤니티, 뉴스레터
- 트렌드를 단순 따라가는 것이 아닌 선별적 수용
- 브랜드 아이덴티티와 트렌드의 균형
```

**Q21: 마감 시 퀄리티와 납기 중 무엇이 더 중요한가요?**
```
답변 팁:
- 이분법적 답변보다는 상황에 따른 판단력 어필
- 소통으로 타협점을 찾는 접근
- 실제 경험 사례 포함
```

**Q22: 우리 회사 제품/서비스를 사용해본 적 있나요?**
```
답변 팁:
- 반드시 미리 사용해보고 면접에 참석
- 구체적 사용 경험과 개선 제안
- "아직 사용 안 해봤어요" 는 준비 부족 신호
```

**Q23: 디자인에서 가장 중요한 요소가 무엇이라 생각하나요?**
```
답변 예시:
- "사용자 관점" (UX 중심)
- "브랜드 일관성" (BI 중심)
- "목적의 명확성" (커뮤니케이션 중심)
지원 직무에 맞는 관점 선택
```

**Q24: 지금까지 받은 가장 어려운 피드백은?**
```
답변 팁:
- 실제 경험 사례
- 처음 느낀 감정 (솔직하게)
- 받아들이고 개선한 과정
- 피드백 이후 성장
```

**Q25: 우리 회사 홈페이지 디자인에 대한 의견은?**
```
답변 팁:
- 반드시 미리 분석해두기
- 칭찬 + 개선 제안 균형
- "저라면 이렇게 했을 것 같아요" 로 창의성 어필
```

### 마무리 질문 (5가지)

**Q26: 연봉은 어느 정도 기대하세요?**
```
답변 팁:
- 미리 시장 연봉 조사 (잡플래닛, 사람인 연봉 정보)
- 범위로 답변 (최소~최대)
- "회사 규정에 따르겠습니다" 도 가능
```

**Q27: 입사 가능 시기는 언제인가요?**
```
답변 팁:
- 교육 수료 일정 확인 후 답변
- 수료 후 2-4주 준비 기간 요청 가능
- 빠를수록 좋다는 인상을 주되, 무리한 약속은 금물
```

**Q28: 다른 곳에도 지원하고 있나요?**
```
답변 팁:
- 솔직하게 답변 (거짓말 금지)
- 이 회사가 1순위임을 강조
- 다른 곳은 회사명을 구체적으로 말하지 않아도 됨
```

**Q29: 마지막으로 하고 싶은 말이 있나요?**
```
좋은 마무리:
- 면접 기회에 감사
- 입사 의지 재확인
- 짧고 임팩트 있게 (30초 이내)
```

**Q30: 이 직무에 지원하는 또 다른 이유가 있나요?**
```
답변 팁:
- 아직 말하지 못한 강점 어필 기회
- 준비해온 내용 추가
- 1-2분 내로 간결하게
```

## 3. 포트폴리오 발표 방법

### 발표 구조 (10분 기준)

```
[오프닝 - 1분]
"안녕하세요, [이름]입니다.
오늘 3가지 주요 작업을 중심으로 발표하겠습니다."

[작업물 1 - 3분]
1. 이 프로젝트를 왜 시작했나? (배경)
2. 어떻게 접근했나? (과정)
3. 결과물 설명 (핵심 디자인 결정)
4. 배운 점 (성장)

[작업물 2 - 3분]
(동일 구조)

[작업물 3 - 2분]
(동일 구조, 더 간결하게)

[클로징 - 1분]
"이 세 가지 작업을 통해 [역량]을 키울 수 있었고,
[회사명]에서 이를 더욱 발전시키고 싶습니다."
```

### 발표할 때 피해야 할 것

❌ "이 작업은 별로 마음에 안 드는데..."
❌ "실력이 부족해서 이렇게 됐어요"
❌ "시간이 없어서 완성을 못 했어요"
❌ 이미지만 보여주고 설명 없음
❌ 툴 이름만 나열 ("포토샵으로 만들었습니다")

✅ 각 디자인 결정에 이유 설명
✅ 어려웠지만 해결한 과정 자랑
✅ 이 경험으로 배운 것 언급
✅ 시선을 면접관과 화면을 오가며 발표

## 4. AI 모의 면접 방법

### Claude 모의 면접 설정

```
지금부터 [회사명] [직무] 1차 면접을 진행해줘.

설정:
- 너: 10년 경력의 디자인팀 팀장
- 나: 신입 디자이너 지원자

규칙:
1. 질문을 하나씩 해줘 (내가 답변할 때까지 기다려)
2. 내 답변에 꼬리질문 1개씩 추가
3. 총 10개 질문 후 종합 피드백
4. 100점 만점 점수와 합격 여부 판단

포트폴리오 요약:
[포트폴리오 주요 작업 목록]

내 자기소개서:
[자기소개서 내용]

시작해줘!
```

### 약점 집중 훈련

```
내 면접 답변 중 부족한 부분을 집중적으로 훈련하고 싶어.

내가 약한 부분:
- "자신의 장단점" 질문에 대한 답변
- 연봉 협상 시 어떻게 말해야 할지
- 역질문을 어떻게 해야 할지

각 부분에 대해 좋은 답변과 나쁜 답변의 예시를 보여주고,
내 답변을 피드백해줘.
```

## 5. 복장·태도 준비

### 복장 가이드

**일반적 기준**:
- **비즈니스 캐주얼 (권장)**: 깔끔한 셔츠/블라우스 + 슬랙스/정장 바지
- **단정한 캐주얼**: 너무 화려하거나 노출이 많은 옷은 피함

**업종별 차이**:
| 업종 | 복장 수준 |
|------|-----------|
| 대기업 | 비즈니스 캐주얼~정장 |
| 광고 대행사 | 비즈니스 캐주얼 |
| 스타트업 | 깔끔한 캐주얼 |
| 출판사/교육 | 단정한 캐주얼~비즈니스 캐주얼 |

**디자이너 특수 팁**:
- 복장 자체로 센스를 보여줄 수 있음
- 단, 지나치게 튀는 것은 피함
- 포트폴리오와 복장의 컬러 톤을 맞추면 좋음

### 면접 태도 체크리스트

**비언어 커뮤니케이션**:
- [ ] 입장 시 밝은 표정으로 인사
- [ ] 적절한 눈 맞춤 (너무 응시하거나 피하지 않기)
- [ ] 바른 자세 유지 (등 기대지 않기)
- [ ] 손은 테이블 위에 자연스럽게
- [ ] 질문 시 살짝 끄덕이며 경청
- [ ] 말할 때 손짓 (적당히)

**언어 커뮤니케이션**:
- [ ] 정중한 존댓말 사용
- [ ] 발음 또렷하게, 적당한 속도
- [ ] "음..." "어..." 최소화
- [ ] 질문 이해 안 되면 "다시 한번 말씀해주실 수 있을까요?" 요청
- [ ] 답변은 결론 먼저, 이유 나중에

## 6. 면접 당일 체크리스트

### 전날 밤

- [ ] 면접 장소·교통편 확인 (네이버 지도로 30분 전 도착 계획)

  - 포트폴리오 (PDF/인쇄본)
