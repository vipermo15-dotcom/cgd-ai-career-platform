# 20. 교육생 실습 - 20주 커리큘럼 종합

## 학습 목표
- 20주 전체 커리큘럼 내용 통합 복습
- 포트폴리오 및 취업 서류 최종 완성
- CGD 플랫폼 모든 기능 숙달
- 취업 준비 완료 상태 확인

## 1. 20주 전체 실습 커리큘럼 요약

### 학기 구성 개요

```
Phase 1: 도구 셋업 (Week 01~05)
├── Week 01: Git/GitHub 기초
├── Week 02: VSCode + Markdown
├── Week 03: Claude 무료 계정 + 프롬프트
├── Week 04: MCP 설치 + GitHub 연동
└── Week 05: Firecrawl 기초 + 기업 조사

Phase 2: 포트폴리오 제작 (Week 06~10)
├── Week 06: 포트폴리오 기획
├── Week 07: 포트폴리오 제작 1
├── Week 08: 포트폴리오 제작 2
├── Week 09: 자기소개서 작성
└── Week 10: 기업분석 + 채용공고 분석

Phase 3: 취업 준비 (Week 11~15)
├── Week 11: 면접 준비 1
├── Week 12: 면접 준비 2
├── Week 13: CGD 플랫폼 활용
├── Week 14: 취업지원 실습
└── Week 15: 포트폴리오 최종 완성

Phase 4: 최종 점검 + 수료 (Week 16~20)
├── Week 16: 자기소개서 최종 수정
├── Week 17: 모의 면접
├── Week 18: 지원서 제출 준비
├── Week 19: 취업 사후 관리
└── Week 20: 졸업 프로젝트 발표
```

## 3. 포트폴리오 완성 체크리스트

### 3.1 필수 포함 항목

```
[ ] 표지 (이름, 직군, 연락처)
[ ] 자기 소개 (1페이지)
[ ] 목차
[ ] 주요 프로젝트 3개 이상
    [ ] 프로젝트 1: 개요 + 제작 과정 + 결과물
    [ ] 프로젝트 2: 개요 + 제작 과정 + 결과물
    [ ] 프로젝트 3: 개요 + 제작 과정 + 결과물
[ ] 보유 스킬 목록
[ ] 수상 및 자격증 (있는 경우)
[ ] 연락처 (이메일, 전화, 링크)
```

### 3.2 디자인 품질 체크

```
[ ] 전체 톤앤매너 일관성 (색상, 폰트, 레이아웃)
[ ] 가독성 (폰트 크기 최소 10pt 이상)
[ ] 여백 활용 (여백이 충분한지)
[ ] 이미지 해상도 (최소 150dpi, 화면용 72dpi)
[ ] PDF 파일 크기 최적화 (20MB 이하)
[ ] 파일명 규칙: 이름_직군_포트폴리오_2026.pdf
```

### 3.3 온라인 포트폴리오 체크

```
[ ] Behance 프로필 완성
    [ ] 프로필 사진
    [ ] 자기소개
    [ ] 최소 3개 프로젝트 업로드
[ ] GitHub 프로필 완성
    [ ] README.md 작성
    [ ] 핀 고정 저장소 설정
[ ] (선택) 개인 포트폴리오 웹사이트
```

## 4. 최종 취업 준비 체크리스트

### 4.1 서류 준비

```
[ ] 이력서 (국문)
    [ ] 사진 부착 (정장, 흰 배경)
    [ ] 인적사항 최신화
    [ ] 학력 사항 정확히 기재
    [ ] 경력/인턴/아르바이트 기재
    [ ] 자격증 기재
    [ ] 포트폴리오 링크 첨부

[ ] 자기소개서
    [ ] 성장 배경 + 지원 동기
    [ ] 학습 경험 + 프로젝트
    [ ] 입사 후 포부
    [ ] 맞춤법 최종 검수 완료
    [ ] A4 기준 1~2페이지 분량

[ ] 포트폴리오
    [ ] PDF 버전 준비 (이메일 제출용)
    [ ] 온라인 링크 준비 (Behance/웹)
    [ ] 인쇄 버전 준비 (대면 면접용)
```

### 4.2 면접 준비

```
[ ] 회사 기본 정보 숙지
    [ ] 설립연도, 직원수, 주요 제품/서비스
    [ ] 최근 뉴스 및 이슈
    [ ] 지원 직무의 주요 업무

[ ] 예상 질문 준비
    [ ] 자기소개 (1분/3분 버전)
    [ ] 지원 동기
    [ ] 포트폴리오 설명
    [ ] 강점/약점
    [ ] 향후 목표
    [ ] 직무 관련 기술 질문

[ ] 면접 당일 준비
    [ ] 복장 (단정한 비즈니스 캐주얼 이상)
    [ ] 포트폴리오 출력물
    [ ] 회사 위치 사전 확인
    [ ] 30분 전 도착 목표
```

### 4.3 디지털 존재감 체크

```

## 5. CGD 플랫폼 활용 가이드

### 5.1 플랫폼 접속 정보

```
URL: https://cgdplatform-pm7dnqip.manus.space
계정: 교수자에게 개별 발급
```

### 5.2 주요 기능 사용 가이드

**취업지원 에이전트 활용**

```
1. 플랫폼 접속
2. "취업지원" 메뉴 클릭
3. 에이전트 선택:
   - 포트폴리오 첨삭 에이전트
   - 자기소개서 작성 에이전트
   - 기업분석 에이전트
   - 면접 준비 에이전트
4. 필요한 자료 업로드 또는 텍스트 입력
5. AI 분석 결과 확인
6. 피드백 반영하여 수정
```

**채용공고 조회**

```
1. "채용공고" 메뉴 접속
2. 직군 필터 선택 (그래픽디자인, UI/UX, 영상 등)
3. 지역 필터 설정
4. 경력 조건 설정 (신입/경력)
5. 관심 공고 북마크 저장
6. 지원 현황 트래킹
```

**기업 DB 검색**

```
1. "기업 정보" 메뉴 접속
2. 기업명 또는 직군으로 검색
3. 기업별 상세 정보 확인:
   - 인재상
   - 주요 사업
   - 채용 트렌드
   - 연봉 정보
   - 복리후생
4. 지원 이력 메모 기능 활용
```

**진로지도 카드 작성**

```
1. "내 프로필" > "진로지도 카드"
2. 6개 항목 입력:
   - 희망 직군
   - 보유 스킬
   - 보완 필요 스킬
   - 목표 기업 유형
   - 예상 취업 시기
   - 특이사항
3. 교수자 공유 설정
4. AI 취업 추천 리포트 받기
```

### 5.3 플랫폼 활용 팁

```
팁 1: 포트폴리오 첨삭은 최소 3회 이상 받기
팁 2: 기업분석 기능으로 목표 기업 5개 이상 저장
팁 3: 채용공고 알림 설정으로 신규 공고 즉시 수신
팁 4: 면접 준비 에이전트로 주 2회 이상 연습
팁 5: 취업 일지 기능으로 진행 상황 지속 기록
```

## 6. 20주 종합 과제 발표 준비

### 발표 구성 (10분)

```
1. 자기 소개 (1분)
   - 이름, 목표 직군, 희망 기업

2. 포트폴리오 소개 (4분)
   - 대표 작품 3개 선정
   - 각 작품의 기획 의도와 제작 과정
   - AI 도구 활용 사례

3. 취업 준비 현황 (2분)
   - 지원 기업 현황
   - AI 도구 활용 성과
   - 앞으로의 계획

4. Q&A (3분)
```

### 발표 자료 구성

```
[ ] PPT 또는 PDF (10~15슬라이드)
[ ] 포트폴리오 PDF (별도 파일)
[ ] 온라인 링크 공유 (Behance/GitHub)
[ ] 취업 준비 현황 표
```

## 7. 수료 후 자기계발 로드맵

```
