\#\# 작업물별 설명 가이드

\#\#\# 작업물 1: 화장품 패키지 시리즈 (3종)  
한 줄 소개: 하나의 브랜드 세계관을 3종 패키지로 일관되게 풀어낸 시리즈 작업  
강조할 것: 컬러·소재·타이포그래피의 통일감 / 고급 질감 표현 / AI로 무드보드·컬러팔레트 도출 과정  
설명 순서: 브랜드 콘셉트 설정 → 컬러·소재 선정 이유 → 3종 전개 → AI 활용 부분 명시  
피할 것: "예쁘게 만들었다"는 주관적 표현 / 브랜드 맥락 없이 결과물만 나열

\#\#\# 작업물 2: 식품 패키지 (전개도 포함)  
한 줄 소개: 실제 인쇄·제작을 고려한 전개도 기반의 식품 패키지 디자인  
강조할 것: 전면·후면·측면 정보 설계 / 식욕 자극 비주얼 / 인쇄 후가공 고려 여부  
설명 순서: 제품 타깃 설정 → 정보 구조 설계 → 비주얼 방향 → 전개도 완성  
피할 것: 전개도 없이 평면 시안만 제출 / 영양성분·바코드 등 필수 정보 누락

\#\#\# 작업물 3: 브랜드 포스터 시리즈  
한 줄 소개: 브랜드 아이덴티티를 포스터 형식으로 확장한 커뮤니케이션 디자인  
강조할 것: 브랜드 톤앤매너 일관성 / 레이아웃 완성도 / AI로 이미지 시안 생성 활용  
설명 순서: 브랜드 선정 이유 → 톤앤매너 분석 → 시리즈 구성 방식 → 결과  
피할 것: 브랜드와 무관한 개인 취향 스타일 적용 / 시리즈 간 톤 불일치

\#\#\# 작업물 4: 광고 콘텐츠 캠페인  
한 줄 소개: 기획부터 비주얼까지 일관된 메시지를 담은 광고 콘텐츠 캠페인  
강조할 것: 기획 의도·타깃·메시지 명확성 / Premiere Pro·AE 활용 영상 포함 시 가산점  
설명 순서: 캠페인 목표 → 타깃 설정 → 크리에이티브 방향 → 결과물 구성  
피할 것: 비주얼만 강조하고 기획 배경 생략 / 단발성 콘텐츠 1장만 제출

\#\#\# 작업물 5: AI 활용 콘셉트 개발 작업물  
한 줄 소개: Gemini·ChatGPT·Claude를 활용해 콘셉트 기획부터 시안까지 제작 과정을 공개한 작업  
강조할 것: AI 도구별 활용 목적 구분 (기획/이미지/텍스트) / 사람의 판단·수정 과정 명시  
설명 순서: 프로젝트 목표 → AI 활용 단계별 설명 → 최종 결과물 → 내가 한 것과 AI가 한 것 구분  
피할 것: AI가 만든 결과물을 그대로 제출 / 프로세스 없이 완성본만 첨부
