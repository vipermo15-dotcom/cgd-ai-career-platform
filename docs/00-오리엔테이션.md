## 1. 플랫폼 소개

### CGD AI Career Platform이란?

CGD AI Career Platform은 서울시기술교육원 컴퓨터그래픽디자인과 교육생이 **AI 도구를 활용하여 취업 준비를 완성**하도록 설계된 통합 학습 플랫폼입니다.

이 플랫폼의 핵심 철학:
> "AI 도구는 보조 수단이다. 최종 결정과 창의성은 반드시 본인의 것이어야 한다."

### 무엇을 배울 수 있나요?

| 분야 | 내용 | 사용 도구 |
|------|------|-----------|
| 도구 기초 | Git, GitHub, VSCode 설치·사용법 | Git, GitHub, VSCode |
| AI 활용 | 프롬프트 작성, Claude/Gemini 활용 | Claude, Gemini, ChatGPT |
| 자동화 | 기업 정보 자동 수집, 채용공고 분석 | Firecrawl, MCP |
| 포트폴리오 | 디자이너 포트폴리오 제작 | Claude, Notion, Behance |
| 자기소개서 | AI 활용 자기소개서 작성·첨삭 | Claude, ChatGPT |
| 기업분석 | 기업 홈페이지 분석, 보고서 작성 | Firecrawl, Claude |
| 면접 준비 | 모의 면접, 포트폴리오 발표 | Claude, Gemini |
| 취업 관리 | 지원 현황 트래킹 | GitHub Projects |

### 이 플랫폼의 특징

**비용 최소화**: Claude 무료 / Gemini 무료 / ChatGPT 무료 / GitHub 무료 / VSCode 무료로 운영 가능

**실무 중심**: 실제 취업 준비에 바로 쓸 수 있는 프롬프트·템플릿·워크플로우 제공

**단계별 설계**: Week01부터 Week20까지 단계적으로 역량을 쌓아가도록 구성

## 2. 사용하는 도구 목록

### 필수 도구 (무료)

#### Git
- **설명**: 파일 변경 이력을 관리하는 버전 관리 시스템 (Version Control System)
- **설치**: [git-scm.com](https://git-scm.com) 에서 무료 다운로드
- **주요 용도**: 파일 저장, 이력 관리, 협업
- **학습 문서**: `docs/01-Git설치.md`

#### GitHub
- **설명**: Git 저장소를 온라인으로 관리하는 클라우드 플랫폼
- **가입**: [github.com](https://github.com) 무료 계정 생성
- **주요 용도**: 포트폴리오 호스팅, 협업, 취업 지원 현황 관리
- **학습 문서**: `docs/02-GitHub.md`

#### VSCode (Visual Studio Code)
- **설명**: 마이크로소프트가 만든 무료 코드·문서 편집기
- **설치**: [code.visualstudio.com](https://code.visualstudio.com) 에서 무료 다운로드
- **주요 용도**: Markdown 문서 작성, Git 연동, 파일 관리
- **학습 문서**: `docs/03-VSCode.md`

#### Claude (claude.ai)
- **설명**: Anthropic이 만든 AI 어시스턴트 - 취업 준비의 핵심 도구
- **가입**: [claude.ai](https://claude.ai) 무료 계정 (Google 계정 연동 가능)
- **주요 용도**: 자기소개서 작성, 기업분석, 면접 준비, 포트폴리오 피드백
- **학습 문서**: `docs/04-Claude무료.md`

#### Gemini (gemini.google.com)
- **설명**: Google이 만든 AI 도구 - 검색·최신 정보에 강점
- **가입**: Google 계정으로 즉시 사용 가능
- **주요 용도**: 최신 채용 정보 검색, 기업 뉴스 분석, 이미지 분석
- **학습 문서**: `prompt/gemini.md`

#### ChatGPT (chatgpt.com)
- **설명**: OpenAI가 만든 AI 도구 - 글쓰기에 강점
- **가입**: [chatgpt.com](https://chatgpt.com) 무료 계정
- **주요 용도**: 자기소개서 초안, 영문 번역, 창의적 글쓰기
- **학습 문서**: `prompt/chatgpt.md`

### 교수자 전용 도구 (유료)

#### Firecrawl
- **설명**: 웹사이트를 자동으로 수집·분석하는 도구
- **비용**: 월 사용료 (교수자가 운영)
- **주요 용도**: 기업 홈페이지 분석, 채용공고 자동 수집, 교육 자료 업데이트
- **학습 문서**: `docs/06-Firecrawl.md`

#### MCP (Model Context Protocol)
- **설명**: AI와 외부 도구를 연결하는 프로토콜
- **설치**: Claude Desktop 설치 후 MCP 서버 연동
- **주요 용도**: GitHub·Firecrawl을 Claude와 연결하여 자동화
- **학습 문서**: `docs/05-MCP.md`

## 4. 첫날 할 일 체크리스트

### 필수 (오늘 반드시 완료)

- [ ] **GitHub 계정 생성**: [github.com](https://github.com) → Sign up
  - 이메일 입력 → 비밀번호 설정 → 사용자명 설정
  - 사용자명은 취업 포트폴리오에 사용되므로 본명 또는 전문적인 이름 추천
  - 예시: `kimdesign`, `parkgraphic`, `leecgd2024`

- [ ] **이 저장소 Fork**: GitHub에서 Fork 버튼 클릭
  - Fork 후 본인 계정의 저장소가 생성됨

- [ ] **Claude 계정 생성**: [claude.ai](https://claude.ai) → 무료 가입
  - Google 계정 또는 이메일로 가입
  - 가입 후 첫 대화 테스트 해보기

- [ ] **Gemini 접속 확인**: [gemini.google.com](https://gemini.google.com)
  - Google 계정 로그인 후 사용 가능

### 권장 (이번 주 내 완료)

- [ ] **Git 설치**: [docs/01-Git설치.md](01-Git설치.md) 참고
- [ ] **VSCode 설치**: [docs/03-VSCode.md](03-VSCode.md) 참고
- [ ] **저장소 Clone**: 터미널에서 `git clone` 명령어 실행

### 오늘의 첫 번째 Claude 대화

GitHub 계정을 만든 후, Claude에게 다음과 같이 첫 대화를 시작해보세요:

```
나는 서울시기술교육원 컴퓨터그래픽디자인과 교육생이야.
취업을 위한 포트폴리오 준비를 시작하려고 해.
내 현재 상황을 설명할 테니 어떤 준비가 필요한지 조언해줘.

현재 상황:
- 디자인 공부 시작한 지 [개월] 됨
- 사용 가능한 툴: [Adobe Photoshop / Illustrator / 기타]
- 목표 직종: [UI/UX 디자이너 / 그래픽 디자이너 / 영상 편집자 / 기타]
- 관심 업종: [광고 / 출판 / IT / 게임 / 기타]
```

## 5. 학습 원칙

### AI 활용의 올바른 자세

**1. AI는 초안을 만들고, 본인이 완성한다**
- AI가 작성한 자기소개서는 반드시 본인 경험으로 수정
- 포트폴리오 내용은 실제 본인 작업물로 구성

**2. 저작권을 지킨다**
- AI 생성 이미지의 상업적 사용 규정 확인
- 다른 사람의 작업물을 포트폴리오에 무단 사용 금지

**3. 개인정보를 보호한다**
- AI 대화에 주민등록번호, 계좌번호 등 민감 정보 입력 금지
- 이 저장소에 API 키, 비밀번호 등 업로드 금지

**4. 꾸준히 기록한다**
- GitHub에 매주 학습 내용 커밋
- 학습 과정 자체가 취업 포트폴리오가 됨

## 6. 질문하는 방법

### 강사에게 질문하기 전에

좋은 질문은 다음 형식을 따릅니다:

```
1. 무엇을 하려고 했나요?
   예: Week01 실습에서 git push 명령어를 실행하려고 했습니다.

2. 어떤 결과가 나왔나요? (오류 메시지 포함)
   예: "fatal: not a git repository" 오류가 발생했습니다.

3. 무엇을 시도해봤나요?
   예: docs/16-문제해결.md를 확인했지만 해결하지 못했습니다.
```

### Claude에게 질문하기

막히는 부분이 있으면 Claude에게 먼저 물어보세요:

```
[상황 설명]
나는 git clone 명령어를 실행했는데 다음 오류가 발생했어.
[오류 메시지 붙여넣기]

어떻게 해결할 수 있어?
```
