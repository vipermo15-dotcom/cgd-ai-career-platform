## 3. 취업 매칭 포인트

### 강점
| 강점 | 활용 방향 |
|---|---|
| K-pop 굿즈 제작이 디자인 입문 계기 | 굿즈디자이너 지원동기의 진정성으로 가장 강력한 차별점 |
| 학생회 홍보국장 — 디자인→발주→제작 전 과정 주도 | 엔터사 디자인팀 공고("기획부터 디자인, 촬영, 제작, 런칭까지 전 과정 참여")의 자격요건과 거의 동일한 경험 |
| GTQ 2급 보유 | 그래픽 툴 활용 능력의 공식 증빙 |
| Figma 활용 가능 | 인디자인·일러스트 외 콘텐츠 기획 도구도 다룰 수 있음을 입증 |
| 학생회 홍보영상 기획·촬영·편집 경험 (패러디 영상, 화제성) | 트렌드 캐치 능력 + 영상 디자인 직무 연결 |
| "처음 쓰는 프로그램도 빠르게 적응" | 경력 4년 미만 갭에 대한 보완 논리 |
| 매장 침수 대응 사례 | 돌발상황 대응력, 다수 이해관계자와의 소통 역량(에이전시·외주업체 커뮤니케이션 조율 업무와 연결 가능) |

### 확인 필요한 갭 (강사 컨펌 필요)
- 디자인 직무 실무경력 0년 → "경력 4년 이상" 요구 공고(예: 유명 엔터사 디자인팀)는 신입 단계에서 즉시 지원하기보다 중장기 목표 공고로 분류 권장
- 인쇄·후가공 프로세스 관련 경험/지식이 자소서·이력서에 언급되지 않음 → 포트폴리오에서 보강 필요
- 지원분야가 이력서별로 다르게 기재되어 있어, 지원 직무에 따라 이력서/자소서를 구분 사용해야 함

## 4. 포트폴리오 구성 전략

### 기본 방향
이력서상 "경력 공백(디자인 실무경력 0년)"을 메우기 위해, **학생회 활동 중 제작한 실제 디자인 결과물**과 **K-pop 굿즈 제작 경험**을 포트폴리오 도입부에 배치하여 경력 대체 자료로 활용.

| 순서 | 작업물 | 연계 포인트 |
|---|---|---|
| 0 (인트로) | K-pop 굿즈 제작 경험(개인 작업) / 학생회 홍보영상(패러디 콘텐츠) | "디자인을 시작하게 된 계기"로 짧게 소개, 경력 공백 보완 자료 |
| 1 | 학생회 포스터·팜플렛·SNS 홍보물 (실물 보유 시) | 기획→디자인→발주→제작 전 과정 주도 경험 — 공고 자격요건과 가장 직접 매칭, 비중 있게 배치 |
| 2 | 캐릭터 굿즈 디자인 시리즈 (실습 작업물) | 자소서 성장과정의 연장선, 전문성 발전 증거 |
| 3 | 굿즈 패키지 디자인 | 인디자인 활용 + 인쇄 공정 이해(보강 필요 항목) 노출 |
| 4 | 숏폼 영상 편집물 | GTQ 2급 + 영상 기획·편집 경험 연계 |
| 5 | AI 활용 SNS 콘텐츠 | 트렌드 이해도 입증용 보조 작업물 |

### 보강 제안 (강사 확인 필요)
1. 학생회 시절 실제 제작한 포스터·팜플렛 원본 파일이 남아있다면 포트폴리오에 우선 포함 — 실무 경력 0년의 공백을 메우는 가장 강력한 자료
2. 인쇄/후가공(박, 형압, 도무송 등) 기초 용어와 공정 이해를 굿즈 패키지 작업물 설명에 1줄씩 추가
3. 자소서 지원동기는 지원 회사·직무에 맞춰 매번 맞춤형으로 재작성 필요 (현재 버전은 특정 엔터사 지원용으로 작성됨)
4. 지원 직무(영상디자이너 vs 굿즈디자이너)에 따라 이력서 버전을 구분해 제출
