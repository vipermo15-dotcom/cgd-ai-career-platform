## 5. 포트폴리오 구성 가이드

### 작업물 5종 구성 (확정)

| 순서 | 작업물 | 유형 | AI 활용 | 주요 취업처 |
|------|--------|------|---------|------------|
| 1 | 숏폼 광고영상 (30초, 패션/뷰티 소재) | 영상 편집 | X or O | 시너윗 ★★★ |
| 2 | SNS 콘텐츠 시리즈 (릴스/쇼츠 3편) | SNS 콘텐츠 | O | 에피소드 ★★★ |
| 3 | AI 활용 광고 비주얼 (제작과정 포함) | 생성형 AI + 편집 | O | 전체 ★★★ |
| 4 | 모션그래픽 / AE 작업 (커머스용) | 모션 | X | 시너윗·에피소드 ★★ |
| 5 | 광고 콘셉트 기획안 (PDF 1장) | 기획·비주얼 | X or O | 돌고래유괴단 ★★ |

### 작업물별 설명 가이드

**작업물 1: 숏폼 광고영상**
- 한 줄 소개: 기획~편집 직접 제작한 패션/뷰티 가상 브랜드 숏폼 광고 (30초)
- 강조할 것: 첫 3초 후킹 장면 선택 이유 + CTA 구성 설명
- 설명 순서: 브리프(배경) → 내 역할 → 편집 포인트 → 결과
- 피할 것: 단순 "편집했습니다" — 왜 이 컷, 왜 이 음악인지 근거 필수

**작업물 2: SNS 콘텐츠 시리즈**
- 한 줄 소개: 릴스/쇼츠 타깃 시리즈 콘텐츠 3편
- 강조할 것: 플랫폼별 포맷 이해 (세로형·자막·후킹) + 시리즈 기획 의도
- 설명 순서: 타깃 설정 → 기획 의도 → 제작 방식 → AI 활용 여부
- 피할 것: 낱개 나열 — 시리즈로 묶어 보여줄 것

**작업물 3: AI 활용 광고 비주얼**
- 한 줄 소개: Midjourney + Photoshop으로 제작한 광고 배너/비주얼
- 강조할 것: 프롬프트 전략 → 편집 과정 스크린샷 → 완성까지 흐름
- 설명 순서: 아이디어 출발점 → AI 프롬프트 → 편집 → 완성
- 피할 것: AI가 만든 것처럼 보이는 나열 — 내 편집·판단 지점 명확히

**작업물 4: 모션그래픽 / AE**
- 한 줄 소개: After Effects로 제작한 커머스용 가격 태그·자막 애니메이션
- 강조할 것: AE 기술 숙련도 + 실제 광고에 바로 쓸 수 있는 실용성
- 설명 순서: 레퍼런스 → 제작 의도 → 기술 포인트 → 완성본
- 피할 것: 기술 과시용 복잡한 이펙트 — 실무 적용 가능성 강조

**작업물 5: 광고 콘셉트 기획안**
- 한 줄 소개: 실존 브랜드 분석 기반 광고 콘셉트 기획 및 비주얼 디렉션
- 강조할 것: 아이디어의 독창성 + 브랜드 분석 근거
- 설명 순서: 브랜드 분석 → 콘셉트 도출 → 비주얼 방향 → 결과물
- 피할 것: 결과물만 올리고 과정 생략 — 사고 과정이 핵심
