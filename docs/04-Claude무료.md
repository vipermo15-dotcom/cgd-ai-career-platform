## 1. Claude란?

Claude는 Anthropic이 만든 AI 어시스턴트입니다.

### 특징

- **한국어 이해도 우수**: 자기소개서, 기업분석 등 한국어 문서 작성에 강점
- **긴 문서 처리 능력**: 채용공고, 기업 홈페이지 내용을 통째로 붙여넣고 분석 가능
- **정확성 중시**: 할루시네이션(잘못된 정보 생성)이 상대적으로 낮음
- **안전 기준**: 개인정보 보호, 윤리적 사용 기준 높음

### 무료 vs 유료 비교

| 기능 | 무료 (Claude.ai) | 유료 (Claude Pro, $20/월) |
|------|-----------------|--------------------------|
| 기본 대화 | 가능 | 가능 |
| Claude 3.5 Sonnet | 제한적 | 무제한 |
| 파일 업로드 | 가능 | 가능 |
| Projects 기능 | 가능 | 가능 |
| 이미지 분석 | 가능 | 가능 |
| 사용량 제한 | 있음 (시간당 제한) | 많음 |

> **교육생 권장**: 무료 계정으로 충분히 활용 가능. 하루 취업 준비 목적으로는 무료로도 충분합니다.

## 2. 무료 가입 방법

### 단계별 가입 절차

**Step 1: 사이트 접속**
- 브라우저에서 [claude.ai](https://claude.ai) 접속

**Step 2: 계정 생성**
- "Sign up" 클릭
- 다음 중 하나 선택:
  - Google 계정으로 가입 (권장 - 가장 빠름)
  - Apple 계정으로 가입
  - 이메일로 가입

**Step 3: Google 계정으로 가입 시**
1. "Continue with Google" 클릭
2. Google 계정 선택
3. 이름 입력
4. 생년월일 입력 (18세 이상 필요)
5. 약관 동의
6. 가입 완료

**Step 4: 전화번호 인증 (경우에 따라)**
- 문자 인증이 필요한 경우 본인 전화번호 입력
- 수신된 인증 코드 입력

### 가입 후 첫 설정

가입 완료 후 Claude에게 자기소개를 해보세요:

```
안녕하세요, Claude! 저는 서울시기술교육원 컴퓨터그래픽디자인과 교육생이에요.
앞으로 취업 준비를 함께 도와주세요!

제 상황:
- 디자인 전공 교육 중
- Adobe CC 툴 배우는 중
- 목표 직종: 그래픽 디자이너
- 관심 업종: 광고/출판/IT

오늘은 어디서부터 시작하면 좋을까요?
```

## 3. 기본 사용법

### 인터페이스 설명

```
[Claude.ai 화면 구성]

좌측 사이드바
├── 새 대화 시작 버튼 (+)
├── Projects 목록
└── 이전 대화 이력

메인 영역
├── 대화 창
├── 메시지 입력창
└── 파일 첨부 버튼
```

### 메시지 입력 방법

- **Enter**: 줄바꿈 (메시지 전송 아님)
- **Shift+Enter**: 줄바꿈
- **전송 버튼** 클릭 또는 **Ctrl+Enter**: 메시지 전송

### 파일 첨부

지원 파일 형식:
- **문서**: PDF, Word (.docx), 텍스트 (.txt), Markdown (.md)
- **이미지**: JPG, PNG, GIF, WebP
- **코드**: 대부분의 코드 파일

첨부 방법: 입력창 왼쪽 클립 아이콘 클릭 → 파일 선택

**활용 예시**:
```
[채용공고 PDF 첨부 후]
이 채용공고를 분석해줘.
내 이력서와 어떤 부분이 매칭되는지 알려줘.
자기소개서에서 강조해야 할 키워드를 뽑아줘.
```

### 이미지 분석

포트폴리오 이미지를 첨부하여 피드백 받기:

```
[포트폴리오 이미지 첨부 후]
이 포트폴리오 디자인에 대해 피드백해줘.
개선할 점과 잘한 점을 구체적으로 알려줘.
취업용 포트폴리오로 적합한지도 평가해줘.
```

## 4. Projects 기능 활용

### Projects란?

Projects는 하나의 주제에 관련된 대화를 묶어서 관리하는 기능입니다.  
문서를 업로드하면 모든 대화에서 참조할 수 있습니다.

### 취업 준비용 Project 만들기

**프로젝트 생성**:
1. 좌측 사이드바 "Projects" → "New Project"
2. 이름: `[회사명] 취업 준비`
3. 설명: `A회사 디자이너 취업 준비 프로젝트`

**프로젝트에 파일 업로드**:
- 채용공고 (PDF 또는 복사한 텍스트)
- 기업 홈페이지 내용 (Firecrawl로 수집)
- 내 이력서 초안
- 포트폴리오 목록

**모든 대화에서 자동 참조**:
업로드한 파일은 해당 프로젝트의 모든 대화에서 참조 가능합니다.

```
[프로젝트 내 대화]
지금까지 업로드한 채용공고와 내 이력서를 기반으로
자기소개서 지원동기 항목을 작성해줘.
A회사의 비전과 내 역량이 연결되도록 작성해줘.
```

### 추천 Project 구성

| Project 이름 | 업로드 파일 | 주요 용도 |
|-------------|------------|-----------|
| 포트폴리오 작업 | 작업물 이미지, 기획서 | 포트폴리오 피드백 |
| 자기소개서 작성 | 이력서, 채용공고 | 자기소개서 첨삭 |
| 기업분석 | 기업 홈페이지, 뉴스 | 기업별 분석 |
| 면접 준비 | 자기소개서, 예상 질문 | 모의 면접 |

## 5. 취업 준비에 활용하는 방법

### 1) 자기소개서 초안 작성

```
나는 서울시기술교육원 컴퓨터그래픽디자인과 교육생이야.
아래 정보를 바탕으로 자기소개서 지원동기 항목을 1000자로 작성해줘.

[지원 회사 정보]
회사명: OO 디자인 스튜디오
주요 업무: 브랜딩, 편집 디자인
채용공고 키워드: 창의적 사고, Adobe CC 숙련자, 협업 능력

[나의 강점]
- Adobe Photoshop, Illustrator 1년 사용
- 대학 광고 동아리 활동 1년
- 협업 프로젝트 경험 3회

위 정보를 바탕으로 자연스럽고 진정성 있는 지원동기를 작성해줘.
```

### 2) 기업 분석

```
아래 기업 정보를 분석해줘.

[기업 정보 - 홈페이지에서 복사한 내용]
(홈페이지 내용 붙여넣기)

다음 항목으로 정리해줘:
1. 기업 소개 (사업 분야, 주요 제품/서비스)
2. 기업 문화 및 가치관
3. 디자이너에게 요구하는 역량
4. 면접에서 활용할 수 있는 포인트 3가지
5. 예상 면접 질문 5가지
```

### 3) 포트폴리오 피드백

```
내 포트폴리오 구성에 대해 피드백해줘.

[포트폴리오 목록]
1. 카페 브랜드 로고 디자인 (Illustrator)
2. 음식점 메뉴판 편집 (InDesign)
3. SNS 카드뉴스 시리즈 10장 (Photoshop)
4. 앱 UI 와이어프레임 (Figma)

지원하려는 직종: 그래픽 디자이너 (출판사)

이 포트폴리오 구성이 적합한지, 
추가하거나 수정할 내용을 구체적으로 알려줘.
```

### 4) 면접 질문 연습

```
디자이너 1차 면접에서 자주 나오는 질문으로 모의 면접을 진행해줘.

방식: 너가 면접관이 되어서 질문해줘.
내가 답변하면 피드백을 해줘.
10개 질문 후 전체 평가를 해줘.

내 정보:
- 지원 직종: 그래픽 디자이너
- 포트폴리오 특기: 브랜딩, 편집 디자인
- 보유 툴: Adobe CC, Figma
```

### 5) 채용공고 분석

```
아래 채용공고를 분석해줘.

[채용공고 전문 붙여넣기]

다음 항목으로 분석해줘:
1. 핵심 자격 요건 정리
2. 우대 사항 분석
3. 실제로 하게 될 주요 업무
4. 이 회사가 원하는 인재상
5. 내가 강조해야 할 경험과 역량
6. 자기소개서에 반드시 포함할 키워드 10개
```

## 6. 프롬프트 작성 팁

### 기본 원칙

**1. 역할 부여**: Claude에게 역할을 명확히 알려주세요
```
너는 10년 경력의 디자인 분야 채용 담당자야.
내 포트폴리오를 보고 솔직한 피드백을 줘.
```

**2. 구체적인 정보 제공**: 막연한 질문보다 구체적인 상황 설명
```
# 나쁜 예
자기소개서 써줘

# 좋은 예
아래 채용공고에 지원하려고 해.
내 경험과 역량을 바탕으로 지원동기 1000자를 작성해줘.
[채용공고 내용]
[내 경험 목록]
```

**3. 출력 형식 지정**: 원하는 형식을 명확히 요청
```
다음 형식으로 작성해줘:
- 분량: 1000자
- 단락 구성: 도입 + 지원동기 + 역량 + 포부
- 어조: 진솔하고 자신감 있는 어조
```

**4. 단계별 요청**: 복잡한 작업은 단계를 나눠서 요청
```
Step 1: 먼저 채용공고의 핵심 키워드를 뽑아줘
Step 2: 내 이력서에서 매칭되는 부분을 찾아줘
Step 3: 두 정보를 연결하여 자기소개서를 작성해줘
```

### 좋은 프롬프트 구조

```
[역할/맥락]
너는 [역할]이야.
내 상황은 [상황 설명]이야.

[정보 제공]
[관련 정보 붙여넣기]

[요청 사항]
위 정보를 바탕으로 [구체적인 요청]해줘.

[출력 형식]
다음 형식으로 작성해줘:
- [형식 사항 1]
- [형식 사항 2]
```

## 7. 저작권 주의사항

### AI 생성 콘텐츠의 저작권

**자기소개서**: AI가 초안을 작성해도, 반드시 본인 경험으로 수정하세요.  
AI 작성 내용을 그대로 제출하는 것은 허위 기재에 해당할 수 있습니다.

**이미지**: Claude가 이미지를 직접 생성하지는 않지만, Midjourney 등 AI 이미지 도구 사용 시:
- 상업적 사용 가능 여부 확인
- 포트폴리오에 AI 생성 이미지 포함 시 명시 권장

**코드/텍스트**: AI가 생성한 텍스트는 저작권 주장이 어려우나, 윤리적 사용 권장

### 개인정보 보호

다음 정보는 Claude에 입력하지 마세요:
- 주민등록번호
- 계좌번호, 카드번호
- 비밀번호
- 타인의 개인정보

## 8. Claude vs 다른 AI 비교

| 비교 항목 | Claude | ChatGPT | Gemini |
|-----------|--------|---------|--------|
| 한국어 능력 | ★★★★★ | ★★★★☆ | ★★★★☆ |
| 자기소개서 작성 | ★★★★★ | ★★★★☆ | ★★★★☆ |
| 최신 정보 | ★★★☆☆ | ★★★★☆ | ★★★★★ |
| 이미지 분석 | ★★★★☆ | ★★★★☆ | ★★★★★ |
| 긴 문서 처리 | ★★★★★ | ★★★★☆ | ★★★★☆ |
| 무료 사용량 | 보통 | 보통 | 많음 |
| 파일 업로드 | 가능 | 가능 | 가능 |

**권장 활용 전략**:
- **Claude**: 자기소개서 작성, 기업분석, 긴 문서 처리, 면접 연습
- **Gemini**: 최신 채용 정보 검색, 기업 최신 뉴스, 이미지 분석
- **ChatGPT**: 영문 번역, 영어 이력서 작성, 창의적 글쓰기
