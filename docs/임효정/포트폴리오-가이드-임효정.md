# 포트폴리오 기업 맞춤형 가이드 — 임효정

## 취업처 특성 분석

```
취업처 1: 한스스타일 (SNS·인스타그램/숏폼 광고 콘텐츠 PD)
- 업종·규모: 패션·라이프스타일 커머스, SNS 마케팅 콘텐츠 전문
- 선호 포트폴리오 스타일: 감성·트렌드 중심, 인스타그램/숏폼 실전 결과물 위주
- 강조할 역량: 프리미어·애프터이펙트 후반작업력, 포토샵·일러스트 1급 자격, 숏폼 기획-편집 전 과정

취업처 2: (주)콘크리트파머스 (퍼포먼스 숏폼 광고 PD — 기획/촬영/편집/AI, 경력무관)
- 업종·규모: 퍼포먼스 마케팅 기반 숏폼 광고 제작
- 선호 포트폴리오 스타일: 데이터·성과 지향, AI 활용 제작 과정을 명시적으로 드러내는 스타일
- 강조할 역량: 생성형 AI(Midjourney·GPT 등) 활용 제작 프로세스, 숏폼 기획력, 경력무관 채용이라 신입 강점 어필 가능

취업처 3: (주)킬링마리오 (그래픽 디자이너 / 편집·CG 디자이너, 신입·경력, 정규직)
- 업종·규모: 자동차 분야 편집·CG 콘텐츠 제작사
- 선호 포트폴리오 스타일: 깔끔·정보전달 중심, 편집디자인 완성도 위주
- 강조할 역량: 인디자인·포토샵 편집디자인 역량, 브랜드디자인·굿즈제작 경험, 끈기 있게 결과물 완성하는 태도
```

> ※ 위 취업처 정보는 2026년 6월 검색 시점 기준이며, 변동 가능성이 있습니다. 실제 지원 시 최신 공고를 다시 확인해 주세요.

## 이력서 기반 취업 매칭 분석

### 강점 / 약점

```
강점
━━━━━━━━━━━━━━━━━━━━━━━━━━━
· GTQi 일러스트 1급 + GTQ 포토샵 1급 → 핵심 툴 공식 역량 증빙 가능 (신입 약점 보완)
· 포토샵·일러스트 "상" 수준 + 프리미어·애펙·인디자인 보유 → 광고콘텐츠+숏폼+편집디자인 풀스택 가능
· 생성형 AI 4종(Gemini, GPT, Midjourney, Claude) 실사용 → "AI 활용 제작 과정"을 요구하는 채용처에 강점
· 커미션 작업 경험(그림→편집→배치) → 디자인 사고 과정을 설명할 수 있는 스토리
· 자소서 일관 메시지: "레이아웃·색상·타이포그래피로 메시지 전달이 달라진다" → 디자인 철학이 명확함
· 끈기·문제해결 태도(작업 환경 악조건에서도 완성) → 협업/마감 압박이 많은 광고대행사·제작사에 어필 포인트

약점 / 보완 필요
━━━━━━━━━━━━━━━━━━━━━━━━━━━
· 디자인 실무 경력 0건 (이디야커피 인턴은 서비스직, 디자인 무관)
· 커미션 작업 = 개인 창작물에 가까워 광고/브랜드 맥락으로 재가공 필요
· 학력: "수료 예정"(2026.09) → 지원 시점에 따라 명시 필요
· 영상 촬영/카메라 장비 경험 언급 없음 (일부 공고에서 우대 사항)
```

## 작업물별 설명 가이드

```
## 작업물 1: SNS 숏폼 광고 콘텐츠

**한 줄 소개**: 트렌드를 빠르게 캐치해 기획→촬영 소스 구성→편집까지 완성한 SNS 숏폼 광고
**강조할 것**:
  · 한스스타일·콘크리트파머스 지원 시 — 프리미어·애프터이펙트로 진행한 색보정·자막·오디오 믹싱 등 후반작업 디테일
  · 콘크리트파머스 지원 시 — Midjourney 등 생성형 AI로 소스를 만든 뒤 직접 편집·완성한 워크플로우를 구체적으로 명시
**설명 순서**: 배경(어떤 제품/메시지를 위한 숏폼이었는지) → 내 역할(기획·편집 전 과정 직접 수행) → 결과(완성본 + 어떤 톤을 의도했는지) → AI 활용(소스 생성 단계 설명)
**피할 것**: "그냥 트렌드 따라 만들었다"는 식의 근거 없는 설명 / AI가 다 했다는 식의 표현

---

## 작업물 2: 브랜드 SNS 콘텐츠 시리즈

**한 줄 소개**: 하나의 브랜드 톤앤매너를 레이아웃·색상·타이포그래피로 일관되게 풀어낸 SNS 콘텐츠 시리즈
**강조할 것**: "같은 메시지도 레이아웃·색상·타이포그래피에 따라 인상이 달라진다"는 본인의 디자인 철학을 이 작업물에서 직접 증명
**설명 순서**: 배경(어떤 브랜드/메시지를 가정했는지) → 내 역할(레이아웃·컬러·타이포 설계 의도) → 결과(시리즈 전체의 일관성) → AI 활용(해당 시, 카피 초안 등 보조적으로만)
**피할 것**: 시리즈 내 톤이 들쭉날쭉해 보이는 구성 / "예쁘게 만들었다"처럼 의도 없는 설명

---

## 작업물 3: 굿즈 디자인 패키지

**한 줄 소개**: 브랜드 아이덴티티를 실물 굿즈(스티커·엽서·파우치 등)로 확장한 디자인 패키지
**강조할 것**: 킬링마리오(편집·CG) 지원 시 — 인쇄·실물 적용을 고려한 색상·재단선 등 제작 디테일까지 신경 쓴 점
**설명 순서**: 배경(브랜드 콘셉트) → 내 역할(아이템별 디자인 적용 방식) → 결과(실물/목업 이미지) → AI 활용(해당 시)
**피할 것**: 단순 일러스트 나열로만 보이는 구성

---

## 작업물 4: 광고 배너/카드뉴스 세트

**한 줄 소개**: 핵심 메시지를 정리한 뒤 레이아웃·컬러 방향을 설계해 완성한 광고 배너·카드뉴스 세트
**강조할 것**: GTQi 일러스트 1급·GTQ 포토샵 1급으로 증빙되는 기본 제작 역량을, 실전 작업물로 한 번 더 보여주는 역할
**설명 순서**: 배경(메시지 정리 과정) → 내 역할(시안 여러 개 → 최종안 선정 과정) → 결과(최종 세트) → AI 활용(GPT로 카피 초안 등, 보조적 활용만 명시)
**피할 것**: 시안 과정 없이 "한 번에 완성했다"는 식의 설명
```

## 포트폴리오 구성 전략 (이력서 반영 보강안)

```
전략 1 — "경력 0건"을 정면 돌파하지 않고 측면 보완
  → 작업물 설명에 자격증(GTQi 1급)과 AI 활용 과정을 "역량 증빙"으로 배치해 경력 공백을 채움

전략 2 — 자소서 디자인 철학을 표지/인트로에 한 줄로 선언
  → "같은 메시지도 레이아웃·색상·타이포그래피에 따라 인상이 달라진다"는 문장을 포트폴리오 인트로에 배치

전략 3 — 커미션 경험은 "전사" 말고 "입문 계기"로만 짧게
  → 실제 작업물은 광고/브랜드 맥락으로 재가공한 결과물 위주로, 커미션은 배경 설명에만 짧게 사용

전략 4 — 취업처별 강조 순서 다르게
  → 콘크리트파머스 제출용: AI 활용 작업물을 1번으로
  → 킬링마리오 제출용: 편집디자인/브랜드디자인을 1번으로
  → 한스스타일 제출용: 숏폼 영상을 1번으로
  (작업물 풀은 동일, 순서만 취업처 맞춤 재배열)
```
