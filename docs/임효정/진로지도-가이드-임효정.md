## 준비 로드맵

```
1. 단기 (1개월):
   - GTQi 일러스트 1급·GTQ 포토샵 1급 외 영상 관련 자격(GTQi 인디자인 등) 또는 촬영 장비 실습 경험 보완
   - 기존 커미션 작업물을 광고/브랜드 맥락으로 재가공해 포트폴리오용으로 전환 시작

2. 중기 (3개월):
   - 콘크리트파머스 지원용: AI 활용 제작 워크플로우(소스 생성 → 직접 편집·완성)를 구체적으로 보여주는 작업물 1개 이상 완성
   - 한스스타일 지원용: 숏폼/릴스 1개 이상 실전 제작(기획→촬영 소스 구성→편집 전 과정)

3. 포트폴리오 핵심 작업물:
   - SNS 숏폼 광고 콘텐츠 (한스스타일·콘크리트파머스 핵심)
   - 브랜드 SNS 콘텐츠 시리즈 (디자인 철학 증명용)
   - 굿즈 디자인 패키지 (킬링마리오 — 편집·CG 강점 어필)
   - 광고 배너/카드뉴스 세트 (3사 공통 기본 역량 증빙)
```

## 포트폴리오 구성 전략 (이력서 반영 보강안)

```
전략 1 — "경력 0건"을 정면 돌파하지 않고 측면 보완
  → 작업물 설명에 자격증(GTQi 1급)과 AI 활용 과정을 "역량 증빙"으로 배치해 경력 공백을 채움

전략 2 — 자소서 디자인 철학을 표지/인트로에 한 줄로 선언
  → "같은 메시지도 레이아웃·색상·타이포그래피에 따라 인상이 달라진다"는 문장을 포트폴리오 인트로에 배치 →
    이후 모든 작업물 설명을 이 철학의 "증거"로 구조화

전략 3 — 커미션 경험은 "전사" 말고 "입문 계기"로만 짧게
  → 실제 작업물은 광고/브랜드 맥락으로 재가공한 결과물 위주로 싣고, 커미션은 자소서식 한 줄 배경 설명에만 사용

전략 4 — 취업처별 강조 순서 다르게
  → 콘크리트파머스 제출용: AI 활용 작업물을 1번으로
  → 킬링마리오 제출용: 편집디자인/브랜드디자인을 1번으로
  → 한스스타일 제출용: 숏폼 영상을 1번으로
  (작업물 풀은 동일, 순서만 취업처 맞춤 재배열)
```
