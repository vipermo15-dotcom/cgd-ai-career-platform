# ChatGPT 활용 프롬프트 모음

## 사용 방법
ChatGPT (chat.openai.com) 또는 ChatGPT 앱에서 활용하세요.
GPT-4o 모델 사용 시 이미지 첨부 및 분석이 가능합니다.

---

## 1. 포트폴리오 프롬프트 (10개)

### GPT-P01: GPT-4o 이미지 분석 + 피드백
```
Act as a professional art director with 10 years of experience.
Analyze this portfolio image and provide feedback in Korean.

[이미지 첨부]

Evaluate:
1. Visual hierarchy
2. Color scheme
3. Typography usage
4. Overall composition
5. Commercial viability

Score each element (1-10) and suggest specific improvements.
```

### GPT-P02: DALL-E로 포트폴리오 커버 아이디어 생성
```
Generate a professional portfolio cover concept for a Korean graphic designer.

Style: Modern, minimal, professional
Color scheme: [선호 색상]
Elements to include:
- Clean grid layout
- Abstract geometric shapes
- Space for name and title
- Dark/light mode compatible

Describe the concept in detail, then generate the image.
```

### GPT-P03: 포트폴리오 목차 구성 최적화
```
나는 컴퓨터그래픽디자인 신입생으로 [직군] 취업을 준비 중입니다.

GPT-4의 웹 브라우저 기능으로 최신 [직군] 포트폴리오 트렌드를 검색하고,
다음을 제안해주세요:

1. 2026년에 효과적인 포트폴리오 구성 순서
2. 채용 담당자가 가장 먼저 보는 페이지
3. 절대 빼면 안 되는 섹션
4. 최신 트렌드를 반영한 독창적 섹션 아이디어

현재 내 목차: [목차 작성]

피드백과 개선안을 제시해주세요.
```

### GPT-P04: 작업물 설명 영어 번역 + 개선
```
포트폴리오의 영어 설명을 작성해주세요.

원본 한국어 설명:
[설명 텍스트]

요청:
1. 영어로 번역 (자연스러운 영어)
2. Behance 국제 버전에 적합하게
3. 150 words 이내
4. 디자인 전문 용어 적절히 사용
5. SEO에 유리한 키워드 포함 (#해시태그 10개 제안)

영어가 서툰 한국 학생도 이해할 수 있도록 단어 설명도 추가해주세요.
```

### GPT-P05: 포트폴리오 구성 A/B 테스트
```
포트폴리오 두 가지 구성 중 어느 것이 더 효과적인지 판단해주세요.

구성 A: [목차 A]
구성 B: [목차 B]

지원 직군: [직군]
목표 기업 유형: [대기업/스타트업/에이전시]

판단 기준:
1. 채용 담당자의 주목 효과
2. 직군 적합성
3. 차별화 효과
4. 리스크 (너무 튀는 것 vs 너무 평범한 것)

어느 구성을 추천하는지, 이유와 함께 알려주세요.
```

### GPT-P06: 포트폴리오 캡션 작성 (작업 설명)
```
각 작업물 옆에 들어갈 캡션을 작성해주세요.

작업 1: [작업명]
- 배경: [왜 만들었나]
- 도구: [사용 도구]
- 결과: [무엇을 달성했나]

캡션 요건:
- 50~80자 (한국어 기준)
- 한 눈에 들어오게
- "이 디자인이 왜 의미 있는가" 전달
- 전문적이지만 읽기 쉽게

3가지 버전 작성 후 최선 추천.
```

### GPT-P07: 수상/자격증 없을 때 대처
```
포트폴리오에 수상 경력이나 자격증이 없습니다.
이를 보완하는 전략을 알려주세요.

현재 상황:
- 공모전 수상: 없음
- 자격증: 없음 (ACA 준비 중)
- 관련 인턴/실습: 없음
- 학교 성적: 중상위권

이 상황에서:
1. 포트폴리오로 역량을 어필하는 방법
2. 자기소개서에서 수상/자격증 공백을 커버하는 방법
3. 면접에서 이 질문이 나올 때 답변 전략
4. 지금 당장 시작할 수 있는 '스펙 대체 전략'
```

### GPT-P08: 포트폴리오 발표 스크립트
```
면접에서 포트폴리오를 발표하는 3분 스크립트를 작성해주세요.

발표할 작업:
[작업 설명]

스크립트 구성:
1. 오프닝 (30초): 주목을 끄는 한 문장으로 시작
2. 배경 (30초): 왜 이 작업을 했는가
3. 과정 (60초): 어떻게 접근했는가 (어려움 + 해결)
4. 결과 (30초): 무엇을 만들고 배웠는가
5. 클로징 (30초): 이 경험이 이 회사에서 어떻게 연결되는가

발표하면서 자연스럽게 "저도 이 회사에서 일하고 싶다"는 의지가 느껴지게.
```

### GPT-P09: 온라인 vs 오프라인 포트폴리오
```
온라인(링크)과 오프라인(인쇄) 포트폴리오를 언제 어떻게 사용해야 하는지 알려주세요.

내 현황:
- PDF 포트폴리오: [완성도]
- Behance: [있음/없음]
- 개인 웹사이트: [있음/없음]

상황별 가이드:
1. 서류 접수 시: 어떤 형식으로?
2. 1차 면접 시: 가져가야 하나?
3. 2차(임원) 면접 시: 어떻게 활용?
4. 취업박람회 참가 시
5. 채용 담당자에게 이메일 발송 시

각 상황별 최적 형식과 준비 방법.
```

### GPT-P10: GitHub Pages로 포트폴리오 만들기
```
GitHub Pages를 활용하여 무료 포트폴리오 웹사이트를 만드는 방법을 알려주세요.

나의 기술 수준: HTML/CSS 기초 知識 있음 / 없음
목표: 포트폴리오를 URL로 공유할 수 있게

단계별 가이드:
1. 저장소 생성 방법
2. 기본 HTML 템플릿
3. 내 포트폴리오 내용 넣기
4. GitHub Pages 활성화
5. 커스텀 도메인 연결 (선택)

실제로 복사해서 쓸 수 있는 HTML 코드도 포함해주세요.
```

## 2. 자기소개서 프롬프트 (10개)

### GPT-J01: ChatGPT로 맞춤법 검사
```
다음 자기소개서의 맞춤법과 문법을 교정해주세요.

[자기소개서 전문]

교정 항목:
- 맞춤법 오류
- 띄어쓰기 오류
- 문장 부호 오류
- 어색한 문장 표현
- 중복되는 단어/표현

교정된 버전과 함께 변경 사항 목록을 표로 제공해주세요.
```

### GPT-J02: 글쓰기 스타일 개선
```
이 자기소개서의 글쓰기 스타일을 더 프로답게 개선해주세요.

현재 자기소개서:
[자기소개서]

개선 방향:
- 수동태 → 능동태로 변환
- 추상적 표현 → 구체적으로
- 진부한 표현 제거 (열심히, 최선, 노력 등)
- 임팩트 있는 첫 문장으로 변경
- 전체 톤 통일

개선 전 / 개선 후 비교 형식으로 보여주세요.
```

### GPT-J03: 분야별 자기소개서 버전 생성
```
하나의 경험으로 다른 직군 자기소개서를 생성해주세요.

공통 경험:
[주요 경험 1~3가지]

버전 A: 브랜드 디자이너 지원용
버전 B: UI/UX 디자이너 지원용
버전 C: 콘텐츠 디자이너 지원용

각 버전에서:
- 강조하는 역량이 달라야 함
- 사용하는 키워드가 직군에 맞아야 함
- 분량: 각 600자

세 버전을 함께 볼 수 있게 표 형식으로 비교해주세요.
```

### GPT-J04: 영문 자기소개서 작성
```
영문 자기소개서를 작성해주세요.

지원 정보:
- 기업: [기업명 (외국계 또는 영어 사용 기업)]
- 직군: [직군]
- 내 배경: [경험 요약]

영문 자기소개서 요건:
- Cover Letter 형식
- 분량: 300~400 words
- 격식체 비즈니스 영어
- 강점 + 지원 동기 + 기여 계획

한국식 영어 문장 피하기
미국/영국 실무 영어 스타일로
```

### GPT-J05: 자기소개서 첫 문장 임팩트 강화
```
자기소개서의 첫 문장을 채용 담당자가 읽고 싶어지도록 바꿔주세요.

현재 첫 문장:
"안녕하세요, 저는 [이름]입니다. 컴퓨터그래픽디자인을 전공한..."

이런 시작은 너무 흔합니다.

더 임팩트 있는 첫 문장 10개를 제안해주세요:
- 질문으로 시작하는 버전
- 강한 주장으로 시작하는 버전
- 구체적 숫자/성과로 시작하는 버전
- 이야기로 시작하는 버전
- 철학/신념으로 시작하는 버전 (각 2개씩)
```

### GPT-J06: 자기소개서 문체 변환
```
같은 내용을 다른 문체로 변환해주세요.

원문:
[자기소개서 섹션]

변환 요청:
A) 공식적 문어체 (대기업 스타일)
B) 세미 포멀 (스타트업/IT 기업 스타일)
C) 스토리텔링 (광고/에이전시 스타일)

어떤 문체가 어떤 기업에 어울리는지 설명과 함께.
```

### GPT-J07: 단어 선택 최적화
```
자기소개서에서 더 강력한 단어를 사용할 수 있도록 최적화해주세요.

현재 사용 중인 약한 표현들:
- "참여했습니다" → ?
- "노력했습니다" → ?
- "했습니다" → ?
- "중요합니다" → ?
- "좋은 결과를 냈습니다" → ?

각 표현을 대체할 강하고 구체적인 동사/표현 5개씩 제안해주세요.
채용 담당자의 눈에 띄는 단어 위주로.
```

### GPT-J08: 역량별 STAR 작성
```
STAR 기법으로 역량별 자기소개서 소재를 작성해주세요.

역량: 문제 해결 능력

내 경험:
[경험 설명]

STAR 형식:
- S (Situation): 어떤 상황이었나
- T (Task): 내가 해야 했던 것
- A (Action): 구체적으로 한 행동
- R (Result): 수치화된 결과

완성된 STAR 단락을 300자로 작성해주세요.
"협업 능력", "창의성", "성실함" 버전도 추가로 작성해주세요.
```

### GPT-J09: 자기소개서 약점 보완
```
자기소개서에서 약점이 될 수 있는 부분을 찾아 보완해주세요.

자기소개서:
[자기소개서 전문]

약점 분석:
1. 채용 담당자가 의심할 수 있는 부분
2. 증거가 없는 주장
3. 너무 일반적인 내용
4. 기업과 맞지 않는 표현

각 약점별 보완 방법과 수정된 문장을 제시해주세요.
```

### GPT-J10: 공백 기간 설명
```
자기소개서나 면접에서 공백 기간을 어떻게 설명할지 도와주세요.

공백 기간 정보:
- 기간: [기간]
- 실제 한 것: [한 것들]
- 공백의 이유: [이유 (솔직하게)]

전략:
1. 공백 기간을 긍정적으로 전환하는 프레이밍
2. 자기소개서에 쓸 수 있는 표현
3. 면접에서 질문 받을 때 30초 답변
4. 절대 하면 안 되는 표현

진솔하면서도 전략적인 설명 방법.
```

## 3. 기업분석 프롬프트 (10개)

### GPT-C01: 기업 SWOT 분석
```
[기업명]을 취업 준비생 관점에서 SWOT 분석해주세요.

분석:
S (Strengths): 이 기업의 강점 (취업자로서 장점)
W (Weaknesses): 이 기업의 약점 (취업자로서 주의할 점)
O (Opportunities): 성장 기회 (내가 기여할 수 있는 부분)
T (Threats): 위협 (이 기업의 위험 요소)

표 형식으로 정리하고,
이 분석을 바탕으로 자기소개서/면접에서 활용할 인사이트 3가지 제안.
```

### GPT-C02: 기업 문화 코드 분석
```
[기업명]의 문화 코드를 분석해주세요.

분석 근거:
- 기업 공식 사이트 내용: [내용]
- 뉴스/보도자료: [내용]
- 직원 리뷰 (블라인드 등): [내용]

분석 항목:
1. 이 기업의 핵심 가치 Top 3
2. 이 기업에서 성공하는 사람의 특징
3. 이 기업에서 힘든 사람의 특징
4. 면접에서 문화 적합성을 보여주는 방법
5. 입사 전 나에게 맞는지 확인할 방법
```

### GPT-C03: 디자이너로서 이 기업 평가
```
[기업명]을 그래픽 디자이너 관점에서 평가해주세요.

평가 항목:
1. 디자인 자율성 (얼마나 창의적으로 일할 수 있는가)
2. 포트폴리오 성장 가능성 (좋은 작업물을 만들 수 있는가)
3. 디자인 팀 규모와 문화
4. 기술 스택 (어떤 도구를 쓰는가)
5. 디자이너로서 5년 후 내 모습

점수(10점) + 총평 + 지원 추천 여부.
```

### GPT-C04: 경쟁 기업 매핑
```
내가 지원할 직군에서 주요 기업들을 매핑해주세요.

직군: [직군]
지역: 서울/수도권

매핑 기준:
- X축: 연봉 수준
- Y축: 성장 기회
- 버블 크기: 기업 규모

각 분면에 해당하는 기업들을 배치하고,
CGD 졸업생이 처음 도전하기 좋은 기업 Top 5 추천.
```

### GPT-C05: 업무 환경 사전 파악
```
[기업명]의 업무 환경을 사전에 파악해주세요.

파악하고 싶은 정보:
1. 재택근무 가능 여부 (현재 정책)
2. 야근 문화 (업계 평균 대비)
3. 도구 지원 (Adobe CC 라이센스, 맥 지급 등)
4. 팀 구성 (디자이너 몇 명인지)
5. 입사 후 OJT/온보딩 방식

이 정보를 면접 역질문으로 활용하는 방법도 알려주세요.
```
