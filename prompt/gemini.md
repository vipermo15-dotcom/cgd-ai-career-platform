# Google Gemini 활용 프롬프트 모음

## 사용 방법
Google Gemini (gemini.google.com) 또는 Google AI Studio에서 활용하세요.
Gemini는 Google Search와 연동되어 최신 정보 검색에 강합니다.

---

## 1. Google Search 연동 활용 프롬프트 (10개)

### G-S01: 최신 채용공고 검색
```
Search the web and find the latest graphic designer job postings in Korea (2026).

Focus on:
- Companies hiring entry-level designers
- Required skills: Adobe CC, Figma, AI tools
- Location: Seoul, Incheon, Gyeonggi

Summarize in Korean with: company name, job title, key requirements, deadline.
Show at least 5 results.
```

### G-S02: 기업 최신 뉴스 검색
```
[기업명]의 최근 6개월 뉴스를 검색하여 정리해주세요.

특히 다음 항목을 포함해주세요:
- 신사업/신제품 발표
- 조직 변화 (인수합병, 구조조정)
- 디자인/브랜딩 관련 뉴스
- 채용 관련 소식

취업 면접 준비에 활용할 수 있도록 핵심만 5가지로 요약해주세요.
```

### G-S03: 디자인 트렌드 최신 정보
```
Search for the latest graphic design trends in 2026.

Include:
- Color trends (Pantone, trending palettes)
- Typography trends
- AI design tools trending
- Portfolio presentation trends
- Social media design trends

Summarize for a Korean design student preparing for job applications.
```

### G-S04: 연봉 정보 검색
```
[직군] 신입 연봉 정보를 검색해주세요.

검색 범위:
- 블라인드 직장인 커뮤니티 정보
- 사람인/잡코리아 연봉 데이터
- 2025~2026년 최신 데이터

포함 항목:
- 신입 평균 연봉 범위
- 기업 규모별 차이 (대기업/중견/중소/스타트업)
- 지역별 차이
- 연봉 협상 성공 사례

표 형식으로 정리해주세요.
```

### G-S05: 경쟁사 디자인 분석
```
[기업명] 디자인 포트폴리오와 브랜드 아이덴티티를 웹에서 검색하여 분석해주세요.

분석 항목:
- 최근 리뉴얼된 로고 또는 BI
- 주력 색상 팔레트
- 타이포그래피 스타일
- SNS 콘텐츠 디자인 방향
- 경쟁사 대비 차별화 포인트

이 기업에 포트폴리오를 제출할 때 어울리는 스타일 제안도 해주세요.
```

### G-S06: AI 디자인 도구 최신 정보
```
Search for the latest AI tools for graphic designers in 2026.

Include:
- Adobe Firefly updates
- Midjourney latest version features
- Canva AI features
- New tools launched in 2025-2026
- Comparison: which tool is best for portfolio work

Korean designers' reviews and usage tips preferred.
```

### G-S07: 포트폴리오 플랫폼 트렌드
```
최신 디자이너 포트폴리오 플랫폼과 트렌드를 검색해주세요.

검색 항목:
- Behance 최신 알고리즘 변화
- 포트폴리오 웹사이트 인기 툴 (2026)
- 채용 담당자들이 선호하는 포트폴리오 형식
- 한국 디자인 업계에서 선호하는 제출 방식
- PDF vs 링크 vs 인쇄물 현황

취업 준비생에게 최적의 포트폴리오 제출 전략을 제안해주세요.
```

### G-S08: 디자인 공모전 정보
```
현재 지원 가능한 국내 디자인 공모전을 검색해주세요.

조건:
- 2026년 상반기 접수 중
- 신입/학생 참가 가능
- 상금 또는 취업 연계 혜택 있는 것
- 그래픽/브랜드/UI 디자인 분야

표로 정리: 공모전명, 주최, 마감일, 상금, 특이사항
포트폴리오에 추가하기 좋은 공모전 Top 3 추천
```

### G-S09: 취업 성공 사례 검색
```
CGD/그래픽디자인 전공자의 취업 성공 사례를 검색해주세요.

검색 키워드 조합 활용:
- "디자인 신입 합격 후기"
- "그래픽 디자이너 취업 성공"
- "포트폴리오 합격 팁"

수집 항목:
- 합격자들의 공통 특징 3가지
- 포트폴리오 구성의 성공 패턴
- 자기소개서에서 유효했던 접근법
- 면접 합격 팁

허구나 광고성 내용 필터링하고 실제 사례 위주로.
```

### G-S10: 해외 취업 정보 (선택)
```
한국 그래픽 디자이너의 해외 취업 가능성을 검색해주세요.

검색 범위:
- 일본 디자인 회사 한국인 채용 현황
- 싱가포르/홍콩 디자인 스튜디오
- 원격 근무 가능한 해외 디자인 직무
- 해외 취업 비자 요건

현실적인 가능성과 준비사항을 정리해주세요.
```

## 2. 이미지 분석 프롬프트 (포트폴리오) - 10개

### G-I01: 포트폴리오 이미지 전체 평가
```
이 포트폴리오 이미지를 분석해주세요.

[이미지 업로드]

평가 항목:
1. 색상 조화 (0~10점)
2. 타이포그래피 선택 (0~10점)
3. 레이아웃 균형 (0~10점)
4. 전체 완성도 (0~10점)
5. 브랜드 일관성 (0~10점)

개선이 필요한 구체적인 요소와 수정 방법을 알려주세요.
```

### G-I02: 색상 팔레트 분석
```
이 디자인 작업물에서 사용된 색상 팔레트를 분석해주세요.

[이미지 업로드]

분석 항목:
1. 주요 색상 추출 (HEX 코드 포함)
2. 색상 조합의 명칭 (보색, 유사색, 삼각배색 등)
3. 색상이 전달하는 감정/메시지
4. 비슷한 업종의 색상 트렌드 대비 평가
5. 개선 방향 (색상 조정 제안)
```

### G-I03: 경쟁 포트폴리오 비교
```
두 포트폴리오를 비교 분석해주세요.

[이미지 1 업로드: 내 작업]
[이미지 2 업로드: 레퍼런스]

비교 분석:
1. 스타일 차이점
2. 완성도 비교
3. 내 작업이 참고 작업보다 나은 점
4. 내 작업이 보완해야 할 점
5. 내 작업을 레퍼런스 수준으로 올리는 방법

영향을 받으면서도 독창성을 유지하는 방법도 제안해주세요.
```

### G-I04: 로고 디자인 평가
```
제가 만든 로고를 평가해주세요.

[로고 이미지 업로드]

평가 기준:
1. 형태의 단순성과 기억 용이성
2. 다양한 사이즈에서의 가독성 (예상)
3. 컬러/흑백 전환 시 효과
4. 트렌드 반영도
5. 독창성
6. 경쟁 브랜드와의 차별성

전체 점수(10점)와 개선 방향 3가지를 제시해주세요.
```

### G-I05: SNS 콘텐츠 디자인 분석
```

5. 현재 트렌드 반영도
```

```
4. 폰트 크기 인쇄 시 가독성
```

```
같은 콘텐츠로 더 효과적인 레이아웃 방향을 3가지 제안해주세요.
```

## 3. 문서 요약 프롬프트 (10개)

### G-D01: 채용공고 빠른 요약
```
이 긴 채용공고를 30초 안에 읽을 수 있게 요약해주세요.

[채용공고 전문]

요약 형식:
- 기업: [기업명]
- 직군: [직군]
- 핵심 업무: 3가지
- 필수 조건: 3가지
- 마감일: [날짜]
- 지원 여부: ⭕/❌ + 이유 한 줄

5줄 이내로 핵심만!
```

### G-D02: 기업 연간보고서 요약
```
이 기업 연간보고서에서 취업 준비에 필요한 내용만 요약해주세요.

[PDF 업로드 또는 내용 붙여넣기]

필요한 정보:
1. 기업 현재 주력 사업 방향
2. 디자인/브랜드 관련 투자
3. 채용 계획 언급 여부
4. 기업 문화 키워드
5. 면접에서 활용할 수 있는 인사이트

3페이지 내용을 3단락으로 요약.
```

### G-D03: 면접 피드백 정리
```
면접 후 받은 피드백을 정리하고 개선 계획을 수립해주세요.

피드백 내용:
[피드백 내용 붙여넣기]

정리 및 계획:
1. 긍정 피드백 정리
2. 개선 요청 피드백 정리
3. 각 개선 사항별 실행 계획
4. 우선순위 (급한 것부터)
5. 다음 면접 전 완료 목표 체크리스트
```

### G-D04: 업계 보고서 요약
```
이 디자인 업계 보고서를 취업 준비생 관점으로 요약해주세요.

[보고서 내용 붙여넣기]

요약 형식:
- 핵심 트렌드 3가지
- 나의 취업 준비에 영향을 미치는 내용
- 포트폴리오에 반영할 수 있는 인사이트
- 자기소개서에 쓸 수 있는 문장 예시 1개

```

```
자료 1 (기업 정보): [내용]
```
