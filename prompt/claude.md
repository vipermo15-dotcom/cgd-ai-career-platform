# Claude AI 취업 준비 프롬프트 모음

## 사용 방법
각 프롬프트를 복사하여 Claude에 붙여넣고, [ ] 안의 내용을 자신의 상황에 맞게 수정하세요.
claude.ai 또는 Claude Desktop에서 사용 가능합니다.

## 1. 포트폴리오 분석 프롬프트 (10개)

### P-01: 포트폴리오 전체 구성 검토
```
당신은 10년 이상 경력의 크리에이티브 디렉터입니다. 
신입 디자이너의 포트폴리오를 채용 담당자 관점에서 검토합니다.

저는 [직군: 브랜드/UI-UX/콘텐츠/영상] 디자이너 취업을 준비 중인 [이름]입니다.
[학교명] [전공] 졸업 예정입니다.

제 포트폴리오 구성은 다음과 같습니다:
[목차를 상세히 적어주세요]

다음을 분석해주세요:
1. 전체 구성의 강점 3가지
2. 반드시 개선해야 할 약점 3가지  
3. [직군] 취업에 특화된 추가 제안 2가지
4. 채용 담당자가 3분 안에 이 포트폴리오를 볼 때 받을 인상
5. 전체 10점 만점 평가 + 이유

한국어로 구체적이고 실용적으로 답변해주세요.
```

### P-02: 작업물 개별 피드백
```
그래픽 디자인 심사위원 역할을 맡아주세요.

다음 작업물에 대해 피드백해주세요:
- 작업명: [작업명]
- 분야: [브랜딩/패키지/편집/모션/UI 등]
- 제작 기간: [기간]
- 제작 목적: [학교 과제 / 개인 프로젝트 / 공모전]
- 작업 설명: [어떤 작업인지 상세 설명]

평가 기준:
1. 디자인 완성도 (색상, 폰트, 레이아웃, 여백)
2. 컨셉의 명확성
3. 포트폴리오 수록 가치 (취업용으로 적합한가)
4. 제작 과정 설명의 필요성
5. 개선 방향 (구체적으로)

표 형식으로 각 기준별 점수(10점)와 이유를 작성해주세요.
```

### P-03: 포트폴리오 스토리텔링 개선
```
당신은 브랜드 스토리텔링 전문가입니다.

제 포트폴리오 Project 01의 설명을 개선해주세요.

현재 설명:
[현재 작성된 프로젝트 설명]

개선 방향:
- 단순 결과물 나열이 아닌 스토리로 구성
- "왜 이 디자인인가"를 설득력 있게 설명
- 500자 이내로 간결하게
- 채용 담당자가 5초 안에 이해할 수 있게

개선된 설명 3가지 버전을 제안해주세요.
(버전 1: 공식적, 버전 2: 캐주얼, 버전 3: 스토리텔링 중심)
```

### P-04: Behance 업로드용 프로젝트 설명
```
Behance 프로젝트 설명 작성을 도와주세요.

프로젝트 정보:
- 제목: [프로젝트 제목]
- 분야: [분야]
- 배경: [왜 이 프로젝트를 했는가]
- 과정: [어떻게 진행했는가]
- 결과: [무엇을 완성했는가]
- 사용 도구: [Adobe 등]

Behance 프로젝트 설명을 한국어와 영어 각각 작성해주세요.
- 한국어: 200자
- 영어: 150 words
- 해시태그 10개 제안 (영어)
```

### P-05: 포트폴리오 커버 문구 작성
```
포트폴리오 표지에 넣을 한 줄 문구를 만들어주세요.

나에 대한 정보:
- 이름: [이름]
- 직군: [희망 직군]
- 핵심 강점: [강점 2~3가지]
- 디자인 철학: [한 줄로 표현한다면]

다음 스타일별로 각 3개씩 제안해주세요:
1. 간결하고 강한 임팩트
2. 감성적이고 시적인 표현
3. 전문적이고 신뢰감 있는 표현

한국어와 영어 각각 포함.
```

### P-06: 작업물 선별 도우미
```
포트폴리오에 넣을 작업물을 선별해주세요.

제 작업물 목록:
[작업물 목록을 작성: 제목, 분야, 완성도(상/중/하), 개인 선호도]

목표: [직군] 취업용 포트폴리오
수록 목표: 프로젝트 3~5개

다음 기준으로 선별 권고를 해주세요:
1. 반드시 포함해야 할 작업 + 이유
2. 제외를 권고하는 작업 + 이유
3. 개선 후 수록 가능한 작업 + 구체적 개선 방법
4. 만약 새 작업물을 추가한다면? 어떤 것이 필요한가
```

### P-07: UI/UX 포트폴리오 케이스 스터디 작성
```
UX 케이스 스터디 작성을 도와주세요.

프로젝트 배경:
- 앱/서비스명: [이름]
- 해결한 문제: [어떤 문제인가]
- 대상 사용자: [누구인가]
- 내가 한 역할: [전체 / 일부]

작업 과정:
- 리서치 방법: [인터뷰, 설문, 데스크 리서치 등]
- 주요 발견: [인사이트]
- 디자인 결정: [왜 이렇게 디자인했는가]
- 결과 측정: [성과가 있다면]

이 내용을 바탕으로 STAR 형식의 케이스 스터디를 작성해주세요:
- Situation: 상황과 배경
- Task: 해결 과제
- Action: 실행한 행동
- Result: 결과와 배운 점

총 700자 내외, Markdown 형식.
```

### P-08: 포트폴리오 PDF 목차 최적화
```
PDF 포트폴리오의 목차와 페이지 배분을 최적화해주세요.

현재 목차:
[현재 목차 작성]

포트폴리오 정보:
- 직군: [직군]
- 총 프로젝트 수: [개]
- 현재 총 페이지: [페이지]

목표:
- 채용 담당자가 3분 안에 핵심을 볼 수 있게
- 각 프로젝트의 전달력 극대화
- 전체 20~30페이지 적정 범위

다음을 제안해주세요:
1. 최적화된 목차 (순서, 각 섹션 페이지 수)
2. 각 순서의 이유 (앞에 오는 게 더 유리한 이유)
3. 절대 빼면 안 되는 섹션
4. 있으면 좋지만 없어도 되는 섹션
```

### P-09: 포트폴리오 → 면접 연계 준비
```
포트폴리오를 바탕으로 면접 질문과 답변을 준비해주세요.

제 포트폴리오 주요 작업:
1. [작업 1]: [한 줄 설명]
2. [작업 2]: [한 줄 설명]
3. [작업 3]: [한 줄 설명]

지원 기업: [기업명]
직군: [직군]

예상 질문 10개와 각 질문에 대한 추천 답변 방향을 작성해주세요.
특히 포트폴리오를 보면서 "이걸 왜 이렇게 했어요?" 류의 질문에 대비할 수 있게.
```

### P-10: 포트폴리오 맞춤화 (기업별)
```
동일한 포트폴리오를 기업별로 어떻게 다르게 제출해야 할지 알려주세요.

내 포트폴리오: [기본 포트폴리오 구성 설명]

지원 기업 A: [기업명] - [분야]
지원 기업 B: [기업명] - [분야]

각 기업에 맞게:
1. 앞에 배치할 프로젝트 순서
2. 강조해야 할 역량/스킬
3. 추가하거나 빼야 할 내용
4. 자기소개 문구 변경 방향

기업 A용, 기업 B용 각각 맞춤 전략을 제안해주세요.
```

## 2. 자기소개서 작성 프롬프트 (10개)

### J-01: 자기소개서 전체 초안
```
취업 자기소개서 초안을 작성해주세요.

지원자 정보:
- 이름: [이름] (자소서에는 생략)
- 전공: 컴퓨터그래픽디자인
- 졸업 예정: [연도]
- 지원 기업: [기업명]
- 지원 직군: [직군]

나의 경험:
- [경험 1: 어떤 작업, 어떤 성과]
- [경험 2]
- [경험 3]

나의 강점:
- [강점 1]
- [강점 2]

입사 후 하고 싶은 것:
- [구체적인 목표]

자기소개서 항목:
1. 성장 배경 및 지원 동기 (400자)
2. 학습 경험 및 역량 (600자)
3. 입사 후 포부 (400자)

기업 인재상 "[인재상]"을 반영하여 작성해주세요.
자연스럽고 진정성 있게, 신입 지원자답게.
```

### J-02: 지원 동기 섹션 작성
```
자기소개서의 "지원 동기" 부분을 작성해주세요.

기업: [기업명]
직군: [직군]
기업의 주요 서비스/제품: [설명]
내가 이 기업에 끌린 계기: [계기]
이 직군을 선택한 이유: [이유]

조건:
- 300~400자
- 진부한 표현 사용 금지 ("어릴 때부터 디자인에 관심이..." 같은 것)
- 구체적인 경험이나 계기를 중심으로
- 기업의 특성과 나의 목표가 연결되게

3가지 버전 작성해주세요.
```

### J-03: 강점/역량 섹션 작성
```
자기소개서의 "역량 및 강점" 부분을 작성해주세요.

내 핵심 강점:
[강점들 나열]

이를 증명하는 경험:
[경험과 성과 나열]

조건:
- 500~600자
- STAR 형식 활용 (상황-과제-행동-결과)
- 추상적 표현 지양, 구체적 수치나 성과 포함
- [직군] 직무와 연결되는 내용 위주
- 마지막에 입사 후 활용 방향으로 마무리

2가지 버전 작성:
버전 1: 기술/실무 능력 중심
버전 2: 협업/소통 능력 중심
```

### J-04: 자기소개서 첨삭
```
제 자기소개서를 첨삭해주세요.

현재 자기소개서:
[자기소개서 전문 붙여넣기]

지원 기업: [기업명]
지원 직군: [직군]

첨삭 항목:
1. 맞춤법/문법 오류
2. 어색한 표현 수정
3. 삭제하면 좋은 내용 (너무 일반적이거나 임팩트 없는 부분)
4. 추가하면 좋은 내용
5. 기업 인재상과 맞지 않는 부분
6. 전체 점수 (10점 만점) + 종합 평가

수정된 전체 버전도 함께 제공해주세요.
```

### J-05: 자기소개서 키워드 추출
```
다음 채용공고에서 자기소개서에 반드시 포함해야 할 키워드를 추출해주세요.

채용공고 내용:
[채용공고 전문 붙여넣기]

추출 요청 사항:
1. 직무 키워드 Top 10 (자주 등장하는 단어)
2. 인재상 관련 키워드 5개
3. 우대 조건 관련 키워드 5개
4. 회사 가치관/문화 키워드 5개
5. 자기소개서에 이 키워드를 자연스럽게 녹이는 예시 문장 3개

각 키워드의 중요도(상/중/하)도 표시해주세요.
```

### J-06: 포트폴리오 제출 이메일 작성
```
포트폴리오를 이메일로 제출할 때 보내는 본문을 작성해주세요.

상황:
- 지원 기업: [기업명]
- 채용 담당자 이름: [이름] (모르면 "인사담당자님")
- 지원 직군: [직군]
- 포트폴리오 링크: [URL]
- 첨부 파일: [파일명]

이메일 요소:
- 제목: (예: "[지원] 그래픽 디자이너 - 홍길동")
- 본문: 200자 내외 (짧고 임팩트 있게)
- 마무리: 면접 기회 요청

격식체(공식적)와 준격식체(세미 포멀) 각각 작성해주세요.
```

### J-07: 자기소개서 분량 조정
```
자기소개서 분량을 조정해주세요.

현재 자기소개서:
[자기소개서 전문]

현재 분량: [xx]자
목표 분량: [xx]자 (기업 요구사항)

분량 [늘리기/줄이기] 작업:
- 핵심 내용은 유지하면서
- 자연스럽게 조정
- 문체와 톤 일관성 유지
- 삭제/추가할 부분 표시

조정된 버전과 함께 어떤 부분을 변경했는지 설명해주세요.
```

### J-08: 자기소개서 면접 연계 예상 질문
```
이 자기소개서를 기반으로 면접관이 할 수 있는 날카로운 질문을 뽑아주세요.

자기소개서:
[자기소개서 전문]

역할: 면접관

예상 질문 유형:
1. 자기소개서 내용 검증 질문 (5개) - "자소서에 ~~라고 했는데 구체적으로?"
2. 약점을 파고드는 질문 (3개)
3. 기업 인재상 관련 질문 (3개)
4. 예상치 못한 돌발 질문 (2개)

각 질문에 대한 추천 답변 방향도 함께 제시해주세요.
```

### J-09: 기업별 맞춤 수정
```
동일 자기소개서를 다른 기업용으로 수정해주세요.

원본 자기소개서 (A기업용):
[자기소개서]

B기업 정보:
- 기업명: [기업명]
- 업종: [업종]
- 인재상: [인재상]
- 규모: [대기업/중견/스타트업]

B기업용으로 수정:
- 지원 동기 부분 수정
- 강조할 역량 변경
- 분위기와 톤 조정 (대기업=공식적, 스타트업=도전적)
- 변경한 부분을 색으로 표시하거나 [수정] 태그 달기
```

### J-10: 자기소개서 최종 검토 체크리스트
```
제 자기소개서 최종 제출 전 전체 검토를 해주세요.

자기소개서:
[자기소개서 전문]

지원 기업: [기업명]
```
